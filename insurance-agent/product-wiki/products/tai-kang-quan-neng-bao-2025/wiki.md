## 1. 产品概览
泰康全能保2025是一款两全保险，集一般身故/高残、多重意外保障和满期生存金返还于一体。特色是意外保障额度高达200万，满期80岁可领取生存金，适合追求保障+储蓄双重功能的客户。

## 2. 保障责任

| 保障项目 | 保障金额 | 给付条件 |
|----------|----------|----------|
| 一般身故/高残 | 基本保额 | 因病身故或高残 |
| 一般意外身故/高残 | 2×基本保额 | 意外伤害导致 |
| 驾乘意外身故/高残 | 10×基本保额 | 驾乘意外伤害 |
| 特定交通工具意外 | 10×基本保额 | 特定交通工具意外 |
| 客运列车意外 | 10×基本保额 | 客运列车意外 |
| 航空意外 | 10×基本保额 | 航空意外 |
| 重大自然灾害意外 | 10×基本保额 | 重大自然灾害意外 |
| 生存保险金 | 6.2×基本保额 | 80岁满期仍生存 |

**注**: 各项保障不累加给付，同一事故按最高一项给付，给付后合同终止。

### 案例(30岁男性)
| 项目 | 数值 |
|------|------|
| 基本保额 | 20万元 |
| 保险期间 | 保至80周岁 |
| 缴费期间 | 20年 |
| 年交保费 | 4200元 |
| 一般身故/高残 | 20万 |
| 意外身故/高残 | 40万 |
| 驾乘/航空等意外 | 200万 |
| 80岁生存金 | 12.4万 |

## 3. 投保规则
- **投保年龄**: 以实际条款为准(案例中30岁投保)
- **保险期间**: 保至80周岁
- **缴费期间**: 20年(以案例为准)
- **等待期**: 以条款为准
- **其他权益**: 现金价值、保单贷款、保险费自动垫交、减保

## 4. 核心卖点
1. **意外保障极高**: 驾乘/航空等意外保障达基本保额10倍(200万)
2. **满期返还**: 80岁满期领取生存保险金，相当于强制储蓄
3. **保障全面**: 一般身故+多种意外场景全覆盖
4. **泰康医养生态**: 可对接泰康医养社区资源
5. **多项权益**: 保单贷款、减保等灵活权益

## 5. 条款要点
- 同一事故多项保障不累加，按最高一项给付
- 满期生存金为6.2倍基本保额
- 可附加健康险产品(医疗险、重疾险等)
- 未成年人身故保险金有限制
- 宽限期一般为60天

## 6. 理赔流程
1. **报案**: 拨打95522或通过泰康APP报案
2. **材料**: 死亡证明/伤残鉴定、保险合同、身份证明等
3. **申请**: 受益人提交理赔申请
4. **赔付**: 审核后赔付至受益人账户

**注意事项**:
- 身故保险金由指定受益人领取
- 满期生存金由被保险人领取
- 未成年人身故保额受监管限制
