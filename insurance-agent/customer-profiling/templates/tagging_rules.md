# 客户标签生成规则

## 一、基础属性标签

### 年龄段
| 年龄范围 | 标签 |
|:---|:---|
| 18-25 | 青年期 |
| 26-35 | 成长期 |
| 36-45 | 黄金期 |
| 46-55 | 稳定期 |
| 56-65 | 预退休期 |
| 65+ | 退休期 |

### 职业风险类别
| 类别 | 说明 | 示例 |
|:---|:---|:---|
| 1类 | 办公室/低风险 | IT、教师、公务员、白领 |
| 2类 | 轻体力/中低风险 | 医生、厨师、营业员 |
| 3类 | 体力/中风险 | 建筑工程师、技工 |
| 4类 | 高风险 | 快递员、出租车司机、外卖员 |

### 家庭结构
| 结构 | 判定条件 |
|:---|:---|
| 单身 | 未婚且无子女 |
| 二人世界 | 已婚无子女 |
| 核心家庭 | 已婚有子女 |
| 三代同堂 | 有父母同住或赡养 |
| 单亲家庭 | 离异/丧偶且有子女 |

## 二、保障现状标签

### 保障完善度评分规则
满分10分，按以下维度评分：

| 维度 | 有=得分 | 无=0分 |
|:---|:---|:---|
| 寿险 | 2分 | 0分 |
| 重疾险 | 2分 | 0分 |
| 医疗险 | 2分 | 0分 |
| 意外险 | 2分 | 0分 |
| 养老/教育储蓄 | 2分 | 0分 |

保额充足性修正：已购但保额低于建议值50%的，该维度减1分。

### 缺口率计算
```
缺口率 = (建议总保额 - 已有总保额) / 建议总保额 x 100%
```

| 缺口率 | 标签 |
|:---|:---|
| 0-20% | 保障充足 |
| 21-50% | 轻度缺口 |
| 51-80% | 中度缺口 |
| 81-100% | 严重缺口 |

## 三、风险偏好标签

基于以下因素综合判定：

| 指标 | 保守型 | 稳健型 | 进取型 |
|:---|:---|:---|:---|
| 已购产品类型 | 纯保障型为主 | 保障+储蓄均衡 | 投资连结/万能为主 |
| 保费占收入比 | <5% | 5-15% | >15% |
| 沟通中表达 | 关注保本、确定性 | 关注性价比、平衡 | 关注收益、灵活性 |

## 四、生命周期标签

| 阶段 | 判定条件 | 核心需求 |
|:---|:---|:---|
| 单身期 | 未婚无子女 | 自身健康保障、储蓄起步 |
| 新婚期 | 已婚无子女/新婚2年内 | 双方保障、购房储蓄 |
| 育儿期 | 有18岁以下子女 | 家庭支柱保障、教育金 |
| 成熟期 | 子女已成年、本人45-60岁 | 养老规划、资产保全 |
| 养老期 | 60岁以上/已退休 | 医疗保障、传承规划 |
| 传承期 | 高净值+关注财富传递 | 保险金信托、资产隔离 |

## 五、交互行为标签

### 活跃度
| 等级 | 判定条件 |
|:---|:---|
| 高 | 近6个月互动>=3次 且 响应率>=70% |
| 中 | 近6个月互动1-2次 或 响应率50-69% |
| 低 | 近6个月互动0次 或 响应率<50% |

### 偏好渠道
取最近5次互动中出现最多的渠道。

---

## 六、意向等级标签

| 等级 | 判定条件 |
|:---|:---|
| 高 | 主动咨询产品 + 近期有明确需求表达 + 响应率>=70% |
| 中 | 有产品咨询记录 + 响应正常 |
| 低 | 仅被动接受关怀 + 无明确需求表达 |

触发节点记录：记录导致意向变化的关键事件（如体检异常、家人生病、子女出生、保单到期等）。
