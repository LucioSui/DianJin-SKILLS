# 保障缺口计算参考手册

本文件为 insurance-gap-advisor 技能提供详细的计算公式、参考数据和参数说明。所有数据均来源于行业通用标准，仅供分析参考。

---

## 一、寿险缺口计算

### 1.1 核心公式

```
寿险缺口 = 家庭责任总额 - 可抵扣资产

家庭责任总额 = A + B + C + D + E
  A = 负债总额（房贷余额 + 车贷余额 + 其他负债）
  B = 子女教育金（从当前年龄到完成学业的总费用）
  C = 父母赡养费（赡养人数 × 年费用 × 预计赡养年限）
  D = 配偶生活保障（年生活费 × 保障年限，通常10-15年）
  E = 丧葬及过渡费用（通常计10-20万）

可抵扣资产 = F + G
  F = 流动资产（存款 + 可变现投资）
  G = 已有寿险保额（含团险）
```

### 1.2 参数取值指引

**子女教育金参考（B）：**

| 教育阶段 | 公立 | 私立 | 国际 |
|---------|------|------|------|
| 幼儿园(3年) | 3-6万 | 10-20万 | 30-60万 |
| 小学(6年) | 6-12万 | 30-60万 | 80-150万 |
| 初中(3年) | 3-6万 | 15-30万 | 40-80万 |
| 高中(3年) | 3-6万 | 15-30万 | 40-80万 |
| 大学(4年) | 8-15万 | 20-40万 | 100-200万 |
| **合计** | **23-45万** | **90-180万** | **290-570万** |

计算时根据子女当前年龄，减去已完成阶段的费用。

**父母赡养费参考（C）：**

假设赡养至预期寿命（通常设为 80-85 岁）：

| 赡养标准 | 年费用 | 说明 |
|---------|-------|------|
| 基础 | 1-2万 | 父母有退休金，仅需补充 |
| 标准 | 2-4万 | 父母退休金不足或无退休金 |
| 优质 | 4-8万 | 含医疗护理、生活品质提升 |

分摊比例：如有兄弟姐妹，按约定比例分摊。默认独生子女承担100%。

**家庭生活费参考（D）：**

| 城市级别 | 月支出(无子女) | 月支出(有子女) |
|---------|--------------|--------------|
| 一线 | 8,000-15,000 | 15,000-30,000 |
| 二线 | 5,000-10,000 | 10,000-20,000 |
| 三线 | 3,000-6,000 | 6,000-12,000 |

保障年限选择依据：
- 子女未成年：保障至子女经济独立（通常22岁大学毕业）
- 配偶无收入：建议保障10-15年过渡期
- 配偶有收入：可缩短至5-10年

### 1.3 经验参考值

| 家庭角色 | 经验保额 | 适用场景 |
|---------|---------|---------|
| 单收入家庭支柱 | 年收入 12-15 倍 | 验证公式计算结果的合理性 |
| 双收入主力 | 年收入 8-10 倍 | 与公式结果交叉校验 |
| 双收入次力 | 年收入 5-8 倍 | 保额不宜过低 |

经验值仅用于合理性校验，最终以公式计算结果为准。

## 二、重疾缺口计算

### 2.1 核心公式

```
重疾缺口 = 治疗费用 + 康复费用 + 收入损失 - 已有重疾保额

治疗费用：按城市级别取参考值
康复费用：治疗费用 × 30%-50%
收入损失：年收入 × 康复年限（通常3-5年）
```

### 2.2 治疗费用参考

| 城市级别 | 基础治疗费 | 含进口药/靶向药 | 说明 |
|---------|----------|---------------|------|
| 一线 | 30-50万 | 50-100万 | 医疗资源丰富，费用较高 |
| 二线 | 20-40万 | 40-80万 | 中等水平 |
| 三线 | 15-30万 | 30-50万 | 费用相对较低 |

### 2.3 常见重大疾病费用明细

| 疾病类型 | 治疗费用 | 康复费用 | 总计(不含收入损失) |
|---------|---------|---------|------------------|
| 恶性肿瘤 | 30-100万 | 10-30万 | 40-130万 |
| 急性心肌梗塞 | 10-30万 | 5-10万 | 15-40万 |
| 脑中风后遗症 | 20-50万 | 10-30万 | 30-80万 |
| 重大器官移植 | 50-100万 | 20-50万 | 70-150万 |
| 冠状动脉搭桥 | 10-20万 | 5-10万 | 15-30万 |

### 2.4 建议保额参考

| 客户特征 | 建议保额 | 推理依据 |
|---------|---------|---------|
| 25-30岁，单身 | 50-80万 | 治疗费为主，收入损失占比低 |
| 30-40岁，家庭支柱 | 80-150万 | 需覆盖治疗+3年收入损失 |
| 40-45岁 | 50-100万 | 保费较高，平衡保额与预算 |
| 高收入(年入50万+) | 100-200万 | 收入损失占比大，需充分覆盖 |

## 三、医疗缺口评估

### 3.1 医疗险类型对比

| 类型 | 保额 | 免赔额 | 医院范围 | 年缴保费参考 |
|-----|------|-------|---------|------------|
| 百万医疗 | 200-600万 | 1万 | 公立二级及以上 | 200-1,000元 |
| 中端医疗 | 100-300万 | 0-1万 | 公立+特需 | 2,000-8,000元 |
| 高端医疗 | 1,000万+ | 0 | 公立+私立+海外 | 1万-10万+ |
| 防癌医疗 | 100-300万 | 0 | 公立二级及以上 | 500-2,000元 |

### 3.2 匹配逻辑

| 客户情况 | 推荐类型 | 推理依据 |
|---------|---------|---------|
| 有社保，预算有限 | 百万医疗 | 1万免赔额社保可覆盖，杠杆率最高 |
| 重视就医体验 | 中端医疗 | 特需部直付，免排队 |
| 高净值/常出国 | 高端医疗 | 全球医疗网络 |
| 有健康异常难投保 | 防癌医疗 | 健康告知宽松 |
| 超过55岁 | 惠民保+防癌医疗 | 年龄限制下的可行方案 |

## 五、预算参考体系

### 5.1 双十原则

- 建议保费：家庭年收入的 5%-15%（保守5%-8%，积极10%-15%）
- 保额目标：年收入的 10 倍以上

### 5.2 各险种保费占比

| 险种 | 建议占比 | 说明 |
|-----|---------|------|
| 重疾险 | 40-50% | 保费大头，优先满足保额 |
| 定期寿险 | 15-25% | 杠杆率高，保费低保额高 |
| 医疗险 | 5-15% | 根据选择类型差异较大 |
| 意外险 | 2-5% | 费用最低 |

### 5.3 年龄与保费关系参考

## 六、健康因素影响评估

### 6.1 常见健康异常对投保的影响

| 健康异常 | 重疾险影响 | 医疗险影响 | 应对策略 |
|---------|----------|----------|---------|
| 甲状腺结节 | 除外甲状腺/加费 | 除外甲状腺 | 多家核保对比，选最优结论 |
| 乳腺结节 | 除外乳腺/加费 | 除外乳腺 | BI-RADS 3级以下争取标体 |
| 高血压(轻度) | 加费或标体 | 可能除外 | 控制良好可争取标体 |
| 高血压(中重度) | 拒保或高额加费 | 拒保 | 考虑防癌险替代 |
| 糖尿病 | 拒保 | 拒保 | 防癌险 + 惠民保 |
| BMI > 28 | 可能加费 | 通常标体 | 如实告知，部分产品宽松 |
| 乙肝病毒携带 | 除外肝脏或加费 | 除外肝脏 | 选对乙肝友好的产品 |

### 6.2 核保结论说明

| 结论 | 含义 | 对分析的影响 |
|-----|------|------------|
| 标准体 | 正常承保 | 无影响 |
| 加费 | 保费上浮 | 需调整预算计算 |
| 除外 | 特定部位/疾病不保 | 需评估除外范围的风险敞口 |
| 延期 | 暂不承保 | 标注为待解决项 |
| 拒保 | 无法投保 | 需寻找替代方案 |

## 七、可解释性输出规范

### 7.1 数据来源标注

所有报告中的数值必须标注来源：

| 标注 | 含义 |
|-----|------|
| [用户] | 用户直接提供的数据 |
| [标准] | 行业通用参考标准 |
| [推断] | 基于已知信息推断，可能存在偏差 |
| [经验] | 基于从业经验的估算值 |

### 7.2 推理链要求

每条建议的推理链必须包含：
1. **事实陈述** — 引用用户的具体情况
2. **规则引用** — 使用的计算公式或判断标准
3. **计算过程** — 数值代入和运算步骤
4. **结论推导** — 从计算结果到建议的逻辑跳转
5. **不确定性说明** — 如有推断值，说明可能的偏差范围
