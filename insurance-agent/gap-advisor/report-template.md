# 保险保障缺口分析报告模板

以下为生成专业 DOCX/PDF 报告时的结构模板。使用 docx skill 或 pdf skill 生成文档时，按此结构组织内容。

---

## 报告结构

```
封面
  报告标题：保险保障缺口分析与配置建议报告
  客户姓名：[姓名]
  报告日期：[YYYY-MM-DD]
  免责提示：本报告为保障分析参考，不构成具体产品购买建议

目录

第一章：客户画像摘要
  1.1 基础信息
  1.2 家庭结构
  1.3 财务概览
  1.4 健康状况摘要（如有）
  1.5 现有保障一览（如有）

第二章：现有保障诊断（如用户有保单）
  2.1 保单持仓一览表
  2.2 诊断发现
    - 各项诊断结论（含判断依据和建议动作）
  2.3 诊断小结

第三章：保障缺口分析
  3.1 寿险缺口
    - 计算过程（含公式、参数来源、逐步运算）
    - 结论与建议保额
  3.2 重疾缺口
    - 计算过程
    - 结论与建议保额
  3.3 医疗缺口
    - 评估过程
    - 结论与建议方向
  3.4 意外缺口
    - 评估过程
    - 结论与建议保额
  3.5 缺口总览表

第四章：配置建议方案
  4.1 配置原则说明
  4.2 逐条建议（每条含完整推理链）
    - 建议内容
    - 推理链（事实→计算→判断→选型→预算）
    - 替代方案
  4.3 方案总览表
    | 险种 | 建议类型 | 建议保额 | 预估年缴保费 | 优先级 |
  4.4 预算分析
    - 总保费与收入占比
    - 各险种分配占比

第五章：实施路径
  5.1 优先级排序（含排序理由）
  5.2 投保时间建议
  5.3 健康告知提醒（如有健康异常）

第六章：动态调整建议
  6.1 需重新评估的触发条件
    - 家庭结构变化（结婚/生子/离婚）
    - 收入重大变化（涨薪/失业/创业）
    - 负债变化（新增房贷/还清贷款）
    - 健康状况变化
  6.2 建议复查周期

附录
  A. 计算公式与参数说明
  B. 参考数据来源
  C. 数据来源标注说明（[用户]/[标准]/[推断]/[经验]）
  D. 免责声明

免责声明全文：
  本报告基于您提供的个人信息和行业通用参考数据生成，旨在提供保障缺口
  分析和配置方向参考。本报告不构成对任何具体保险产品的购买建议或推荐。
  保险产品的选择应结合个人实际情况，建议在专业保险顾问指导下进行投保决策。
  报告中的保费预估基于市场平均水平，实际保费以保险公司核保结果为准。
  标注为[推断]的数据基于合理假设，可能与实际情况存在偏差。
```

## 格式规范

### 表格样式

所有数据表格使用以下样式：
- 表头：深蓝背景(#1F4E79)，白色字体，加粗
- 奇数行：白色背景
- 偶数行：浅蓝背景(#D6E4F0)
- 边框：细灰色边框(#BFBFBF)

### 计算过程展示

使用缩进和箭头展示计算过程，例如：

```
寿险缺口计算：

  家庭责任总额：
    房贷余额                    100万  [用户]
  + 子女教育金(公立，13年)        35万  [标准]
  + 父母赡养费(2人×3万×15年)     90万  [用户+标准]
  + 家庭生活费(20万×10年)       200万  [用户]
  + 过渡费用                     15万  [标准]
  ────────────────────────────────────
  = 总需求                      440万

  可抵扣资产：
    流动资产                     50万  [用户]
  + 已有寿险保额                 30万  [用户]
  ────────────────────────────────────
  = 总可抵扣                     80万

  寿险缺口 = 440万 - 80万 = 360万
  建议保额 = 360万 × 1.1(安全边际) ≈ 400万
```

### 推理链展示

使用编号和缩进展示推理链：

```
【建议】配置定期寿险，保额400万，保至60岁

【推理链】
  ① 事实：用户为单收入家庭支柱，38岁，房贷100万，
     子女5岁，配偶全职主妇
  ② 规则：寿险缺口 = 家庭责任总额 - 可抵扣资产
  ③ 计算：缺口 = 440万 - 80万 = 360万（详见3.1节）
  ④ 判断：上浮10%安全边际 → 建议保额400万
  ⑤ 选型：定期寿险杠杆率最优，保至60岁覆盖子女
     成年及房贷还清
  ⑥ 预算：预估年缴约7,200元，占年收入1.4%

【替代方案】
  - 预算紧张：保额降至300万，年缴约5,400元
  - 保障更充分：增加至500万，年缴约9,000元
```

### 缺口总览表格式

```
| 保障维度 | 建议保额 | 已有保额 | 缺口 | 缺口率 | 紧迫度 |
|---------|---------|---------|------|-------|-------|
| 寿险    | 400万   | 30万    | 370万 | 92.5% | 高    |
| 重疾    | 100万   | 0       | 100万 | 100%  | 高    |
| 医疗    | 百万医疗 | 社保    | 补充  | -     | 高    |
| 意外    | 100万   | 0       | 100万 | 100%  | 中    |
```

### 优先级标注

| 优先级 | 标识 | 含义 |
|-------|------|------|
| 高 | 红色标记 | 存在重大保障缺口，建议立即配置 |
| 中 | 橙色标记 | 保障不足，建议3个月内配置 |
| 低 | 绿色标记 | 可选配置，根据预算决定 |
