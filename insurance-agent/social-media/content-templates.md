# 小红书笔记文案模板

## 风格一：干货科普型

**适用场景**：政策解读、保险知识普及、社保新规解读

**标题模板**（20字以内，带emoji）：

```
[数字]+[核心关键词]+[利益点/情绪词]
```

**标题示例**：
- `医保改革后这3点变化，99%的人不知道`
- `5个社保冷知识，第3个直接省下大几千`
- `养老金怎么领最划算？看完这篇就懂了`

**正文模板**：

```
[1句话点题，引发好奇]

[emoji] [要点1标题]
[要点1详细说明，2-3句]

[emoji] [要点2标题]
[要点2详细说明，2-3句]

[emoji] [要点3标题]
[要点3详细说明，2-3句]

[总结金句，1句话收尾]

[emoji] 觉得有用记得收藏+关注，持续分享保险干货~

#保险知识 #[具体标签1] #[具体标签2] #保险科普 #[个人IP标签]
```

**完整示例**：

标题：`医保改革后这3点变化，打工人必看`

正文：
```
2025年医保个账改革全面落地，很多人还不知道自己的医保卡发生了什么变化

1. 门诊可以报销了
以前门诊花费只能用个人账户，现在纳入统筹基金报销，报销比例50%-70%，看门诊终于不用全自费了

2. 个人账户里的钱变少了
在职职工个人账户只计入个人缴费部分，单位缴费不再划入。简单说，卡里每月到账的钱会减少

3. 个人账户可以给家人用了
配偶、父母、子女都可以共用你的医保个人账户余额，买药、看门诊都行

划重点：门诊能报销是好事，但个人账户缩水意味着日常医疗支出要自己兜底的部分变多了。建议配置一份补充医疗险，把缺口补上

觉得有用记得收藏+关注，持续分享保险干货~

#医保改革 #社保知识 #补充医疗险 #保险科普 #保险顾问
```

## 风格二：热点关联型

**适用场景**：社会热点事件、时事新闻、季节性话题

**标题模板**：

```
[热点事件关键词]+[反转/警醒/行动点]
```

**标题示例**：
- `看完这条暴雨新闻，我立刻加了涉水险`
- `又一个年轻人猝死，你的保险能赔吗？`
- `刷到这条热搜后，我连夜给爸妈买了保险`

**正文模板**：

```
[热点事件引入，场景感强，1-2句]

[emoji] 事件背后值得关注的点：
[分析角度1，与保险/风险相关]
[分析角度2，给出具体数据或事实]

[emoji] 这件事给我们的启示：
[从事件过渡到保障意识，2-3句]
[给出具体的保障建议，不推具体产品]

[emoji] 生活有风险，保障要提前准备
有问题随时评论区聊~

#[热点标签] #[保险相关标签1] #[保险相关标签2] #风险防范 #[个人IP标签]
```

**完整示例**：

标题：`看完这条暴雨新闻，赶紧查了下车险`

正文：
```
刚刷到新闻，某地突发暴雨，地下车库被淹，上百辆车泡水。评论区一片哀嚎，好多车主才发现——自己的车险根本不赔

这里有两个很多人不知道的盲区：

1. 车损险改革后，涉水导致的发动机损坏已纳入保障范围。但如果你的保单是旧版的，可能没有这项

2. 车辆泡水后二次点火导致的损失，保险公司是拒赔的。切记泡水后千万别启动车辆，直接打救援电话

给大家的建议：
翻出你的车险保单，确认下是否包含发动机涉水保障。马上到续保期的朋友，记得关注这一项

天灾没法控制，但我们可以提前做好保障规划

有车险相关问题欢迎评论区交流~

#暴雨天气 #车险知识 #涉水险 #车辆保障 #保险避坑
```

## 风格三：真实案例型

**适用场景**：理赔案例、客户故事、保险价值体现

**标题模板**：

```
[人物身份]+[事件]+[保险结果/金额]
```

**标题示例**：
- `90后妈妈确诊甲癌，重疾险一次赔了30万`
- `一份意外险，救了一个四口之家`
- `住院15天花了8万，保险报了7万6`

**正文模板**：

```
[故事开头，代入感强的场景描写，2-3句]

[emoji] 事情经过：
[案例核心情节，保险在其中的作用，3-4句]

[emoji] 理赔结果：
[具体数据：赔了多少、多久到账、解决了什么问题]

[emoji] 这个案例给我的感触：
[总结感悟，传递温度，不硬推销，2-3句]

愿每个家庭都有一份安心的底气

#真实理赔 #[险种标签] #[情感标签] #保险故事 #[个人IP标签]
```

**完整示例**：

标题：`90后宝妈确诊甲癌，3年前的决定救了她`

正文：
```
前几天看到一个理赔案例，特别想分享给大家

一位90后年轻妈妈，体检时发现甲状腺结节，进一步检查确诊甲状腺乳头状癌早期

她3年前给自己买了一份重疾险，当时预算不多，选了个基础版，每年保费不到3000块

确诊后，保险公司审核通过，一次性赔付30万

她说这笔钱让她可以安心治疗休养，不用担心房贷断供，也不用让老公一个人扛所有压力

"当时只是觉得年轻买便宜，随手就买了，没想到真的用上了"

有时候保障的意义，不是锦上添花，而是在风雨来临时，给你一个不慌张的底气

愿每个家庭都有一份安心的保障

#真实理赔 #重疾险 #甲状腺癌 #家庭保障 #保险故事
```

## 小红书文案写作要点

### 标题公式（选择其一）

| 公式 | 示例 |
|------|------|
| 数字法 | "3个社保冷知识，第2个90%的人不知道" |
| 疑问法 | "养老金到底怎么算？终于有人讲清楚了" |
| 对比法 | "有保险vs没保险，住一次院差距太大了" |
| 场景法 | "住院15天后，我才明白保险有多重要" |
| 清单法 | "30岁前一定要买的4种保险（附避坑指南）" |

### 正文排版规范

- 每段不超过3行，段与段之间空一行
- 使用emoji作为段落分隔和项目符号（常用：1. 2. 3. 或自定义emoji列表）
- 重要内容用【】或「」框起来强调
- 正文末尾放3-5个话题标签

### 话题标签规范

- 每条笔记配3-5个标签
- 第1个标签：内容核心主题（如 #医保改革）
- 第2个标签：保险品类（如 #重疾险）
- 第3个标签：受众群体或场景（如 #打工人必看）
- 第4-5个标签：个人IP或长尾词（如 #保险顾问 #保险科普）

### 禁用词汇

以下词汇不得出现在文案中：

- "保证收益""绝对安全""稳赚不赔"
- "再不买就晚了""错过就没了"
- "全网最低""限时特价""手慢无"
- "某某公司不好""XX产品垃圾"
- 具体的保险产品名称和费率（除非用户明确要求）
- "点击链接""复制口令"等引流词
