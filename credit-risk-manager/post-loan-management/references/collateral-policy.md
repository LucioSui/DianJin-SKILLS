# 担保有效性核查政策

## 担保类型核查标准

| 担保类型 | 核查内容 | 合格标准 | 预警阈值 |
|---------|---------|---------|---------|
| 房产抵押 | 市场价值、他项权证有效性、物理状态 | 抵押率 ≤ 70%，他项权证在有效期内 | 价值下跌 ≥ 15%、房产被查封、闲置 |
| 土地抵押 | 土地价值、使用权期限、规划限制 | 抵押率 ≤ 60%，剩余期限 ≥ 授信期限 | 规划调整、土地闲置 > 1 年 |
| 设备抵押 | 评估价值、成新率、技术迭代 | 抵押率 ≤ 50%，成新率 ≥ 40% | 技术淘汰、成新率 < 40%、闲置 |
| 应收账款质押 | 付款义务人资信、账期、确认函 | 质押率 ≤ 80%，付款方 AA 级以上 | 付款方评级下调、账期延长、纠纷 |
| 保证担保 | 保证人财务、征信、担保余额 | 保证人资产负债率 ≤ 70%，无逾期 | 保证人自身经营恶化、担保过度 |
| 存货质押 | 市场价值、保险、监管状态 | 质押率 ≤ 60%，已投保财产险 | 价格下跌、保险到期、监管失控 |

## 保证人过度担保判断
- 对外担保 / 净资产 > 50% → 预警
- 保证人资产负债率 > 70% → 预警
- 保证人出现逾期或评级下调 → 预警

## 担保物红色预警条件
- 担保物被司法查封或扣押 → 一票否决条件（P4）
- 关键权证遗失/失效导致担保悬空 → 一票否决条件（P4）
- 担保物价值跌幅 ≥ 25% → 红色预警
