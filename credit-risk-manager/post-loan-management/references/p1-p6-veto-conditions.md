# P1-P6 一票否决条件

触发以下任一条件的，视为重大信贷风险，须立即红色预警并启动应急报告和处置。

| 编号 | 一票否决条件 | 处置要求 |
|------|-------------|---------|
| P1 | 贷款资金被挪用至房地产投机、股市、期货、民间借贷等禁止性领域，且客户拒绝整改或无法限期整改 | 红色预警，启动提前收贷 |
| P2 | 客户发生重大诉讼/仲裁且涉案金额超过净资产 30%，或作为被告的诉讼标的合计超过净资产 50% | 红色预警，评估资产保全措施 |
| P3 | 实际控制人失联超过 15 天、发生刑事犯罪被羁押、死亡或丧失民事行为能力，且无合格接替人 | 红色预警，启动应急接管 |
| P4 | 担保物被司法查封/扣押/拍卖且本行优先受偿权受损，或关键权证遗失/失效导致担保悬空 | 红色预警，启动资产保全 |
| P5 | 客户发生交叉违约（对其他金融机构本金或利息逾期超过 90 天），或征信出现连续 3 次及以上逾期记录 | 红色预警，压缩敞口 |
| P6 | 企业生产经营实际停止（主要生产线停工超过 3 个月、主要资质被吊销、营业执照被注销） | 红色预警，移交不良资产 |

## 上报路径
- 红色预警：报行长、董事会风险委员会，必要时监管报备
- 时限：2 个工作日内完成上报
