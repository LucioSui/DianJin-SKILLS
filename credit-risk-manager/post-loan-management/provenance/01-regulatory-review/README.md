# 步骤 1：监管文件研读

## 输入
- 《商业银行金融资产风险分类办法》（银保监会 中国人民银行令〔2023〕年第1号）
- 《贷款风险分类指引》（银监发〔2007〕54号）
- 《商业银行信用风险内部评级体系监管指引》（银监发〔2008〕67号）
- 内部贷后管理制度文件

## 过程
1. 研读监管文件中的贷后管理要求，提取贷后检查频率、方式、内容等监管底线要求
2. 分析风险分类（五级分类）的核心定义和调整原则
3. 识别监管文件中的强制性要求（如关注类以上必须实地检查、逾期 90 天以上至少归为关注等）
4. 对照原 Skill 中的检查频率矩阵和分类标准，确认是否符合最新监管要求

## 输出
- 监管要求清单（共 12 项强制性要求）
- 与原版 Skill 的合规差距分析
