# 步骤3: 监管合规分析

## 输入
- 监管政策文件(发改委、工信部、金融监管总局、生态环境部)
- 产业结构调整指导目录(发改委)

## 过程
- 梳理通用监管要求(工商登记、环保合规、安全生产)
- 梳理行业特色监管要求(农业/制造业/建筑/商贸/服务业)
- 设计监管合规要求参考文件(regulatory-requirements.md)
- 跟踪近1-3年专项整治重点和新增准入要求
- 设计行业生命周期判定标准(industry-lifecycle-guide.md)

## 输出
- 监管合规要求参考references/regulatory-requirements.md(62行)
- 行业生命周期判定标准references/industry-lifecycle-guide.md(51行)
- 近1-3年专项整治重点清单(环保整治、安全整治、税务整治)
- 新增准入要求清单(碳排放权交易、数据安全合规、反垄断审查)

## 关键决策
- 每个文件标注data_valid_until日期(2026-08-01)
- 环保敏感行业强制要求排污许可证核查
- 生命周期阶段与信贷策略映射(初创期谨慎介入、成熟期积极介入)
