# 步骤2: 专家访谈与案例萃取

## 输入
- 欺诈模式案例库(银行信贷案例、监管处罚公告)
- 信贷审查实务指南(银行内部信贷手册、审查标准)

## 过程
- 萃取各行业典型财务造假手法(农业/制造业/商贸/建筑/服务业)
- 归纳每种造假手法的发现线索和反制规则
- 设计欺诈模式目录(fraud-pattern-catalog.md)
- 验证欺诈反制规则覆盖完整性

## 输出
- 欺诈模式目录references/fraud-pattern-catalog.md(70行)
- 5大类行业造假手法及反制规则映射表
  - 农业/养殖类: 虚报养殖规模/产量、关联方交易虚增收入
  - 制造业: 存货虚增/固定资产注水、收入虚增(虚开发票)
  - 商贸流通: 渠道压货虚增收入
  - 建筑施工: 应收账款虚报/少计坏账
  - 服务业: 预付款虚增/关联占款

## 关键决策
- 每种造假手法必须有对应的基础核查反制规则
- 区分造假方向:收入虚增型vs成本压缩型vs资产虚增型
- 发现线索必须含量化阈值(如单位面积产量超行业上限30%)
