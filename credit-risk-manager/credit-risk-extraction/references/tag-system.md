# 企业标签体系参考

> data_valid_until: 2027-01-01

## 行业分类（核心标签）

- **标准**：GB/T 4754-2017 国标行业分类标准
- **格式**：【门类代码】【门类名称】>【大类代码】【大类名称】>【中类代码】【中类名称】>【小类代码】【小类名称】
- **示例**：C制造业 > C36汽车制造业 > C367汽车零部件及配件制造 > C3670汽车零部件及配件制造
- **要求**：必须穿透至小类（四位数代码），不得止于大类或中类

## 基础属性标签

| 标签类型 | 可选值 | 填写说明 |
|---------|--------|--------|
| 企业规模 | 微型 / 小型 / 中型 / 大型 | 参照工信部中小企业认定标准（营收/资产/员工人数） |
| 所有制性质 | 国有企业 / 民营企业 / 外资企业 / 合资企业 | 以实际控制人性质为准 |
| 资本属性 | 上市公司（A股/港股/美股） / 新三板挂牌 / 非上市公司 | **必填**，上市须标注交易所及股票代码 |
| 组织架构 | 集团企业 / 单一企业 / 分支机构 | **必填**，集团须说明层级关系 |
| 生命周期 | 初创期（<3年）/ 成长期 / 成熟期 / 衰退期 | 结合成立年限、营收增速综合判断 |

## 经营特征标签

| 标签类型 | 可选值 | 填写说明 |
|---------|--------|--------|
| 供应链特征 | 国内采购主导 / 进口依赖型 / 全球采购型；供应商集中度（高/中/低） | 高集中度：前三大供应商采购占比 > 50% |
| 生产模式 | 离散制造 / 流程制造 / 混合制造 | 非制造业企业填"不适用" |
| 技术特征 | 高新技术企业 / 专精特新 / 传统制造；自动化程度（智能/高度/部分/人工） | 高新技术须有认定证书支撑 |
| 销售模式 | 直销主导 / 经销主导 / 出口导向 / 内销主导 | 出口占比 > 30% 可标注出口导向 |

## 风险特征标签

| 标签类型 | 可选值 | 填写说明 |
|---------|--------|--------|
| 合规风险 | 安全生产（高/中/低危）；环保合规（重/中/轻污染）；质量合规（强制/一般/无认证） | 高危行业参照安监总局《高危行业目录》 |
| 财务风险 | 资本结构（高/中/低杠杆）；现金流（紧张/健康/充裕）；应收账款（长/中/短账期） | 高杠杆：资产负债率 > 70%；长账期：平均回款 > 180天 |
| 司法风险 | 被执行 / 失信被执行人 / 限制高消费 / 涉诉频发 / 无司法风险 | 通过裁判文书网、企查查等交叉验证 |

## 外部环境标签

| 标签类型 | 可选值 | 填写说明 |
|---------|--------|--------|
| 政策环境 | 政策鼓励类 / 政策中性 / 政策限制类 / 政策淘汰类 | 对照发改委《产业结构调整指导目录》（现行版） |
| 市场环境 | 增量市场（行业增速 > 10%）/ 存量竞争（增速 3%-10%）/ 萎缩市场（增速 < 3%）| 参照行业协会或统计局近3年数据 |

## 标签填写通用规则

1. **有据可依**：每个标签须能在原文或公开信息中找到支撑，不得凭印象填写
2. **必填项不得为空**：资本属性、集团属性（组织架构）为必填，原文未提及填"（需补充核实）"
3. **允许跳过**：非制造业企业的"生产模式"等不适用的标签，填"不适用"而非空白
4. **禁止分级**：风险特征标签的定性描述（高/中/低）须有量化依据，不得凭感觉判断
