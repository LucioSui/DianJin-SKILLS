# 政策指标定义与传导路径模板

> data_valid_until: 2026-08-01

## 货币政策指标

### LPR(贷款市场报价利率)
- **定义**: 商业银行对其最优质客户执行的贷款利率
- **发布机构**: 中国人民银行
- **发布频率**: 每月20日(遇节假日顺延)
- **传导路径**: LPR下调 → 企业融资成本降低 → 信贷需求上升 → 银行信贷规模扩大
- **数据口径**: 1年期和5年期以上两个品种

### MLF(中期借贷便利)
- **定义**: 央行向符合条件的商业银行提供的中期基础货币
- **发布机构**: 中国人民银行
- **传导路径**: MLF利率下调 → 银行负债成本降低 → LPR报价下行 → 实体经济融资成本下降

### 存款准备金率
- **定义**: 金融机构为保证客户提取存款和资金清算需要而准备的在中央银行的存款
- **传导路径**: 降准 → 银行可贷资金增加 → 信贷供给扩大 → 市场流动性改善

## 宏观经济指标

### GDP增速
- **定义**: 国内生产总值同比增长率
- **发布机构**: 国家统计局
- **发布频率**: 季度
- **传导路径**: GDP增速上升 → 企业经营环境改善 → 信贷需求增加 → 银行资产质量提升

### PMI(采购经理指数)
- **定义**: 通过对企业采购经理的月度调查结果统计汇总、编制而成的指数
- **警戒线**: 50%为荣枯线,高于50%反映经济扩张,低于50%反映经济收缩
- **传导路径**: PMI回升 → 企业生产活动活跃 → 短期融资需求上升 → 流动资金贷款增长

### M2增速
- **定义**: 广义货币供应量同比增长率
- **传导路径**: M2增速回升 → 市场流动性充裕 → 银行资金来源充足 → 信贷投放能力增强

## 监管政策指标

### 资本充足率
- **定义**: 商业银行资本总额与风险加权资产之比
- **监管要求**: 系统重要性银行≥11.5%,其他银行≥10.5%
- **传导路径**: 资本充足率下降 → 银行风险资产扩张受限 → 信贷投放增速放缓

### 不良贷款率
- **定义**: 不良贷款余额与贷款总额之比
- **监管要求**: 一般≤2%
- **传导路径**: 不良率上升 → 银行风险偏好下降 → 信贷审批趋严 → 信贷供给收缩

## 国际外部环境指标

### 美联储联邦基金利率
- **定义**: 美国商业银行间隔夜拆借利率
- **发布机构**: 美联储(Fed)
- **传导路径**: 美联储加息 → 中美利差收窄 → 资本外流压力 → 人民币汇率承压 → 出口企业融资需求变化

### 人民币汇率中间价
- **定义**: 中国外汇交易中心每日公布的人民币对美元汇率中间价
- **发布机构**: 中国外汇交易中心
- **传导路径**: 人民币贬值 → 出口企业竞争力提升 → 出口导向型企业信贷需求增加
