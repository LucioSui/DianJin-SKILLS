# 财务压力测试参数

## 压力测试情景设定

### 轻度压力（基准情景）
| 参数 | 设定值 |
|------|--------|
| 收入下降 | -10% |
| 毛利率变化 | -2 pct |
| 产能利用率变化 | -5 pct |
| 应收账款周转天数增加 | +10 天 |

### 中度压力（不利情景）
| 参数 | 设定值 |
|------|--------|
| 收入下降 | -20% |
| 毛利率变化 | -5 pct |
| 产能利用率变化 | -10 pct |
| 应收账款周转天数增加 | +20 天 |

### 重度压力（极端情景）
| 参数 | 设定值 |
|------|--------|
| 收入下降 | -35% |
| 毛利率变化 | -10 pct |
| 产能利用率变化 | -20 pct |
| 应收账款周转天数增加 | +40 天 |

## DSCR 计算口径

```
DSCR = 经营现金流净额 / 当期应还本息
     = (EBITDA - 所得税 - 营运资金变动) / (当期本金 + 当期利息)
```

### DSCR 判断标准

| DSCR 值 | 判断 | 授信建议 |
|---------|------|---------|
| > 1.5 | 充足 | 维持或适度增加 |
| 1.2-1.5 | 关注 | 维持，设定监控指标 |
| 1.0-1.2 | 预警 | 压缩敞口，追加担保 |
| < 1.0 | 不足 | 启动风险处置 |

## 报表口径校验规则

- 合并报表数据只能与合并报表数据对比
- 母公司口径数据只能与母公司口径数据对比
- 若报表口径不清晰，默认使用合并报表假设，并在输出中标注

## 资产减值计提参考

| 资产类别 | 减值测试方法 | 计提参考比例 |
|---------|-------------|-------------|
| 应收账款 | 账龄分析法 | 1年内5%、1-2年20%、2-3年50%、3年以上100% |
| 存货 | 可变现净值法 | 成本与可变现净值孰低 |
| 在建工程 | 减值测试 | 可收回金额低于账面价值的差额 |
| 固定资产 | 减值测试 | 可收回金额低于账面价值的差额 |
