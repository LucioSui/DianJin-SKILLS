# 01 - 监管文件研读

## 输入

- 《商业银行贷款业务管理办法》(银监会令2010年第2号)
- 《商业银行授信工作尽职指引》(银监发〔2004〕51号)
- 《征信业管理条例》
- 银行内部授信业务操作规程

## 过程

### 监管要求提取

| 监管文件 | 核心要求 | 映射到Skill功能 |
|---------|---------|----------------|
| 商业银行贷款业务管理办法 | 贷款申请与受理明确 | 步骤1客户基本信息校验 |
| 授信工作尽职指引 | 授信申请材料完整性要求 | 步骤2材料完整性检查 |
| 征信业管理条例 | 征信查询须取得书面授权 | 步骤2征信授权书校验 |
| 内部授信业务操作规程 | 具体进件材料标准 | 材料分级标准(必备/建议/补充) |

### 一票否决红线定义

基于监管要求,定义6条一票否决红线(I1-I6):
- I1: 企业营业执照已过期、被吊销,或统一社会信用代码无法核验
- I2: 法定代表人身份证过期或无法核验真实性
- I3: 资金用途明确指向禁止领域(股市、房地产投机、理财、非法集资、赌博等)
- I4: 征信授权书缺失或授权范围不足,无法合法获取征信报告
- I5: 企业已被列入经营异常名录或严重违法失信名单,且未解除
- I6: 申请材料存在明显伪造痕迹(如印章异常、文件格式不符、数据前后矛盾)

### 合规检查清单

- [x] 客户基本信息与官方证照逐字比对(步骤1)
- [x] 材料分级准确(必备/建议/补充三级)(步骤2)
- [x] 征信授权书合规性检查(原件扫描件、授权范围覆盖)(步骤2)
- [x] 金额与用途合理性校验(步骤3)
- [x] 一票否决条件前置检查(步骤4)
- [x] 进件结论明确(可进件/不可进件/限期补齐)(步骤5)

## 输出

- 监管要求映射表(4条监管文件→Skill功能映射)
- 一票否决红线定义(I1-I6)
- 合规检查清单(6项)
