# 步骤1:监管文件研读与要求映射

## 输入
- 《商业银行押品管理指引》(银监发〔2017〕16号)
- 《民法典》第二编物权(第二分编 担保物权)
- 《不动产登记暂行条例》及实施细则
- 《应收账款质押登记办法》(中国人民银行令〔2019〕第4号)
- 《动产和权利担保统一登记办法》(中国人民银行令〔2021〕第7号)

## 过程
1. 通读5部监管文件,提取押品管理核心要求
2. 识别押品准入、评估、登记、监控、处置五环节监管红线
3. 定义一票否决条件(C1-C6):
   - C1:押品已被司法机关查封、冻结、扣押
   - C2:押品权属存在争议或重复抵押/质押未披露
   - C3:押品属于法律法规明确禁止抵押或质押的财产
   - C4:评估价值虚高,偏离同类资产市场价超过30%
   - C5:关键权证缺失、伪造或与登记信息不一致
   - C6:保单已失效或保险金额低于评估价值110%
4. 映射监管要求到Skill流程节点

## 输出
- 监管要求映射表(见[规则提取步骤README](../02-rule-extraction/README.md))
- 一票否决条件定义(C1-C6)
- 监管文件编号与条款索引

## 监管要求摘要
| 监管文件 | 核心要求 | Skill映射步骤 |
|---------|---------|--------------|
| 《商业银行押品管理指引》 | 押品准入清单、评估机构资质、登记完备性、存续期监控 | 步骤1-4 |
| 《民法典》担保物权分编 | 抵押权/质权设立、优先受偿权、处置程序 | 步骤3,5 |
| 《不动产登记暂行条例》 | 不动产抵押登记程序、他项权证管理 | 步骤3 |
| 《应收账款质押登记办法》 | 中登网登记规则、付款义务人确认 | 步骤3 |
| 《动产和权利担保统一登记办法》 | 动产质押统一登记、登记期限 | 步骤3 |
