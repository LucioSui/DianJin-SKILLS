# 转债估值与博弈速评

## 技能概述

本技能面向证券固定收益研究员，提供可转债的快速估值分析与博弈机会挖掘。核心功能包括转股溢价率测算、强赎风险评估、下修博弈分析、套利机会识别，输出标准化Markdown格式速评报告。

## 数据获取策略

### 核心数据源：gildata-aidata

所有转债行情、正股数据、公告舆情均优先通过 `gildata-aidata` 服务获取。

| 数据需求 | 推荐工具 | 说明 |
|---------|---------|------|
| **转债价值分析** | `CBBondValueAnalysis` | 获取可转债转股溢价率、纯债溢价率、转股价值、到期收益率、强赎/下修触发价、剩余规模、评级等多维度价值指标 |
| **转债龙虎榜** | `CBDailyTopListDetails` | 获取可转债上榜原因、资金流向、异动区间表现等宏观资金面数据 |
| **正股实时行情** | `AShareLiveQuote` | 获取正股最新价、涨跌幅、成交量、委比、市值等实时行情 |
| **正股估值指标** | `StockValueAnalysis` | 获取正股PE、PB、EV、股息率等估值指标 |
| **正股区间行情** | `StockRangeQuotation` | 获取正股近30日行情走势，用于判断强赎/下修触发趋势 |
| **转债公告** | `BondAnnouncement` | 检索债券市场公告，涵盖强赎、下修、回售、到期等类型 |
| **转债舆情** | `SecurityNewslist` | 查询债券相关舆情，支持按情感类型、证券代码筛选风险信息 |

### 数据获取优先级

1. **转债核心价值**：优先 `gildata-aidata` 的 `CBBondValueAnalysis`，一键获取转债多维度价值指标。
2. **正股行情与估值**：优先 `gildata-aidata` 的 `AShareLiveQuote` 和 `StockValueAnalysis`。
3. **公告与舆情**：优先 `gildata-aidata` 的 `BondAnnouncement` 和 `SecurityNewslist`。
4. **补充信息**：`web_search` 仅用于补充突发新闻细节或交叉验证关键数据。

## 工作流程

### 第一步：获取转债基础数据与价值分析

使用 `finx gildata-aidata CBBondValueAnalysis` 查询目标转债的核心价值指标：

```
finx gildata-aidata CBBondValueAnalysis --body '{
  "query": "[转债名称/代码]"
}'
```

该工具返回可转债收益、转股、纯债、波动率、触发价及基金持仓等多维度价值指标，包括：
- 转债最新价格、涨跌幅
- 转股溢价率、纯债溢价率
- 转股价值、到期收益率
- 转股价、强赎触发价、下修触发价
- 剩余规模、债券评级、到期日
- 波动率指标

### 第二步：获取正股实时行情与估值

使用 `finx gildata-aidata AShareLiveQuote` 查询正股实时行情：

```
finx gildata-aidata AShareLiveQuote --body '{
  "query": "[正股名称/代码]"
}'
```

使用 `finx gildata-aidata StockValueAnalysis` 查询正股估值指标：

```
finx gildata-aidata StockValueAnalysis --body '{
  "query": "[正股名称/代码]"
}'
```

获取正股数据包括：
- 最新股价、涨跌幅、成交量
- 近20日均价、近30日均价
- 市盈率（PE）、市净率（PB）
- 总市值、流通市值

如需正股近期走势，使用 `finx gildata-aidata StockRangeQuotation` 查询区间行情：

```
finx gildata-aidata StockRangeQuotation --body '{
  "query": "[正股名称/代码] 近30个交易日行情"
}'
```

### 第三步：查询转债公告与舆情

使用 `finx gildata-aidata BondAnnouncement` 查询转债相关公告：

```
finx gildata-aidata BondAnnouncement --body '{
  "query": "[转债名称/代码] 强赎 下修 回售 到期"
}'
```

使用 `finx gildata-aidata SecurityNewslist` 查询转债舆情：

```
finx gildata-aidata SecurityNewslist --body '{
  "query": "[转债名称/代码]"
}'
```

### 第四步：核心指标测算

#### 1. 转股溢价率分析

```
转股溢价率 = (转债价格 - 转股价值) / 转股价值 × 100%
转股价值 = 100 / 转股价 × 正股价格
```

**判断标准：**
- 溢价率 < 0%：折价状态，存在套利空间
- 溢价率 0%-20%：低溢价，股性强
- 溢价率 20%-50%：中等溢价，平衡型
- 溢价率 > 50%：高溢价，债性强

#### 2. 强赎风险评估

**强赎触发条件（常见条款）：**
- 正股价格连续30个交易日中至少有15个交易日不低于转股价的130%
- 或转债价格持续高于130元

**风险评估要点：**
- 当前正股价格 / 转股价 比值
- 近30日正股价格走势
- 公司是否有强赎历史
- 公司资金压力与促转股意愿

**强赎风险等级：**
- 🔴 高风险：正股价/转股价 > 1.3，且持续多日
- 🟡 中风险：正股价/转股价 1.2-1.3，接近触发线
- 🟢 低风险：正股价/转股价 < 1.2

#### 3. 下修博弈分析

**下修触发条件（常见条款）：**
- 正股价格连续30个交易日中至少有15个交易日低于转股价的85%（或80%）

**下修可能性评估：**
- 当前正股价 / 转股价 比值
- 剩余期限（临近到期下修意愿更强）
- 公司是否有下修历史
- 大股东持债比例（利益关联度）
- 公司偿债压力

**下修空间测算：**
```
下修后转股价 ≥ 股东大会召开日前20日均价 和 前1日均价 的较高者
理论下修后转债价值 = 100 / 新转股价 × 正股价格
```

#### 4. 套利机会挖掘

**折价套利：**
- 转股溢价率 < 0% 的转债
- 可买入转债 → 转股 → 卖出正股（需考虑T+1交易规则）

**低溢价博弈：**
- 溢价率 < 10% 且正股有上涨预期
- 转债价格 < 110元 且溢价率较低

**下修博弈：**
- 满足下修条件 + 公司有下修意愿
- 剩余规模小、大股东持债比例高

**回售博弈：**
- 临近回售期 + 正股价低于回售触发价
- 公司可能通过拉升股价或下修转股价避免回售

### 第五步：生成速评报告

**理论下修空间：**

## 工具使用规范

1. **数据查询**：优先使用 `gildata-aidata` 服务获取权威金融数据
   - `CBBondValueAnalysis`：可转债多维度价值分析（核心工具）
   - `AShareLiveQuote`：正股实时行情
   - `StockValueAnalysis`：正股估值指标（PE/PB等）
   - `StockRangeQuotation`：正股区间行情（用于判断趋势）
   - `BondAnnouncement`：转债相关公告查询
   - `SecurityNewslist`：转债舆情信息

2. **多源验证**：关键数据可通过 `web_search` 交叉验证

3. **实时性**：标注数据获取时间，提醒用户注意时效性

4. **计算准确**：所有溢价率、比值等指标需精确计算

## 注意事项

1. **条款差异**：不同转债的强赎/下修条款可能不同，需以募集说明书为准
2. **交易规则**：转股套利需考虑T+1规则及交易成本
3. **信用风险**：低评级转债需关注违约风险
4. **流动性**：小规模转债可能存在流动性不足问题
5. **免责声明**：报告末尾必须包含风险提示与免责声明

## 示例调用

用户输入："分析下宁银转债"

技能执行：
1. 获取转债价值：gildata-aidata CBBondValueAnalysis (查询宁银转债)
2. 获取正股行情：gildata-aidata AShareLiveQuote (查询宁波银行)
3. 获取正股估值：gildata-aidata StockValueAnalysis (查询宁波银行PE/PB)
4. 获取转债公告：gildata-aidata BondAnnouncement (查询宁银转债强赎/下修公告)
5. 计算核心指标：转股溢价率、强赎风险等级、下修可能性
6. 生成报告：按模板生成 Markdown 格式转债估值与博弈速评报告
7. 输出报告：直接在对话中显示完整报告内容
