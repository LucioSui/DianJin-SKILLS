# 产业债财报风控体检

## 技能概述

本技能面向证券固定收益研究员的产业债投资与风控场景，提供发债主体全方位财务风险筛查能力。通过自动获取企业财务数据、司法风险、评级变动、担保信息等，系统性识别现金流恶化、有息负债压力、担保链传染、诉讼纠纷等负面信号，输出结构化风控体检报告，辅助债券投资决策与持仓风险管理。

## 工作流程

### 第一步：确认分析主体

获取用户指定的企业名称或债券代码。若用户提供的是债券代码，先通过 `finx gildata-aidata CreditBondBaseInfo` 查询债券对应的发债主体名称。

若主体为非上市公司，需明确告知用户财务数据获取可能受限，将主要依赖工商信息、司法信息、评级信息等进行风险评估。

### 第二步：获取企业基础信息

使用 `finx gildata-aidata CompanyArchives` 查询企业工商基本信息，获取统一社会信用代码及基础资料。

```
finx gildata-aidata CompanyArchives --body '{"companyName": "企业名称"}'
```

若需要查询发债主体特定信息（如存续债券列表），使用 `finx gildata-aidata BasicInfoBondIssuer`。

```
finx gildata-aidata BasicInfoBondIssuer --body '{"issuerName": "企业名称"}'
```

### 第三步：获取财务数据

使用 `finx gildata-aidata` 系列工具查询企业核心财务指标。

1. **财务报表数据**：
```
finx gildata-aidata FinancialStatements --body '{"issuerName": "企业名称", "reportType": "合并", "startDate": "2023-01-01", "endDate": "2026-05-11"}'
```
获取资产负债表、利润表、现金流量表关键科目（如货币资金、短期借款、长期借款、应付债券、经营活动现金流净额等）。

2. **财务分析指标**：
```
finx gildata-aidata BondFinancialAnalysis --body '{"issuerName": "企业名称", "startDate": "2023-01-01", "endDate": "2026-05-11"}'
```
获取盈利能力、偿债能力（流动比率、速动比率、资产负债率、利息保障倍数等）、营运能力、成长能力等指标。

若用户提供了财报 PDF 文件，读取文件内容并提取关键财务数据。

### 第四步：获取司法与负面信息

并行查询以下司法风险信息：

```
# 被执行人信息
finx gildata-aidata Executee --body '{"companyName": "企业名称", "page_index": 1, "page_size": 20}'

# 失信被执行人
finx gildata-aidata DishonestPersonList --body '{"companyName": "企业名称", "page_index": 1, "page_size": 20}'

# 开庭公告
finx gildata-aidata CourtSession --body '{"companyName": "企业名称", "page_index": 1, "page_size": 20}'

# 裁判文书
finx gildata-aidata Judgement --body '{"companyName": "企业名称", "page_index": 1, "page_size": 20}'

# 公司股权出质
finx gildata-aidata CorporateEquityPledge --body '{"companyName": "企业名称", "page_index": 1, "page_size": 20}'

# 限制高消费
finx gildata-aidata LimitHighConsume --body '{"companyName": "企业名称", "page_index": 1, "page_size": 20}'

# 经营异常
finx gildata-aidata AbnormalOperationInfo --body '{"companyName": "企业名称", "page_index": 1, "page_size": 20}'

# 监管处罚/行政处罚
finx gildata-aidata RegulatoryPenaltyList --body '{"companyName": "企业名称", "page_index": 1, "page_size": 20}'

# 欠税公告
finx gildata-aidata TaxArrearsAnnouncement --body '{"companyName": "企业名称", "page_index": 1, "page_size": 20}'

# 对外担保信息
finx gildata-aidata ExternalGuarantee --body '{"companyName": "企业名称", "page_index": 1, "page_size": 20}'
```

### 第五步：获取评级信息

使用 `finx gildata-aidata` 查询主体评级变动。

```
finx gildata-aidata BondIssuerRatingChange --body '{"issuerName": "企业名称", "startDate": "2024-01-01", "endDate": "2026-05-11"}'
```

同时使用 `finx gildata-aidata SecurityNewslist` 搜索企业最新评级变动和负面舆情。

```
finx gildata-aidata SecurityNewslist --body '{"startDate": "2026-05-01", "endDate": "2026-05-11", "securityCode": "主体相关代码或名称"}'
```

或使用 `NewsPublicOpinionList` 检索更广泛的新闻舆情。

```
finx gildata-aidata NewsPublicOpinionList --body '{"query": "企业名称 违约 评级下调 负面", "startDate": "2026-05-01", "endDate": "2026-05-11"}'
```

### 第六步：获取担保与关联信息

通过 `ExternalGuarantee` 获取对外担保明细。
通过 `FinancialStatements` 中的关联方往来款项分析关联风险。
通过 `CorporateEquityPledge` 获取股权质押信息。

## 风险信号判定规则

### 现金流风险信号

| 信号 | 判定条件 | 风险等级 |
|------|---------|---------|
| 经营现金流为负 | 最新报告期经营活动现金流净额 < 0 | 🟠 较高风险 |
| 净现比持续恶化 | 连续两期净现比 < 1 且下降 | 🟡 中等风险 |
| 双负依赖筹资 | 经营+投资现金流均为负 | 🟠 较高风险 |

### 有息负债风险信号

| 信号 | 判定条件 | 风险等级 |
|------|---------|---------|
| 资产负债率超标 | 资产负债率 > 70% | 🟡 中等风险 |
| 短期偿债压力大 | 现金短债比 < 1 | 🟠 较高风险 |
| 付息能力不足 | 利息保障倍数 < 3 | 🟠 较高风险 |
| 期限错配严重 | 短期有息负债占比 > 50% | 🟡 中等风险 |

### 司法风险信号

| 信号 | 判定条件 | 风险等级 |

| 限制高消费 | 存在记录 | 🔴 高风险 |
| 大额被执行人 | 执行标的 > 1000万 | 🟠 较高风险 |
| 股权高比例质押 | 质押比例 > 50% | 🟠 较高风险 |

### 担保链风险信号

| 信号 | 判定条件 | 风险等级 |

## 注意事项

1. **数据时效性**：财务数据以最新可得报告期为准，注意区分年报、中报、季报
2. **非上市公司**：非上市公司财务数据可能不完整，需依赖工商信息和司法信息
3. **集团企业**：对于集团型企业，注意区分母公司与合并口径数据
4. **行业差异**：不同行业的财务指标阈值存在差异，需结合行业特点判断
5. **动态跟踪**：风控体检反映的是某一时点的快照，建议定期复查
6. **信息交叉验证**：重要风险信号建议通过多渠道交叉验证

## 输出要求

- 报告必须使用 Markdown 格式
- 所有数据需标注来源和报告期
- 风险信号必须明确标注等级
- 综合风险评级必须给出明确结论
- 报告生成后直接在对话中显示完整内容
