# 固定收益研究员资金面&利率复盘简报

## 技能概述

本技能面向证券固定收益研究员，自动化生成资金面与利率复盘简报。通过获取银行间市场核心利率数据、分析利率变动逻辑、结合宏观与政策因素，输出结构化的复盘报告。

## 数据获取流程

### 第一步：获取当前日期

使用 `finx time` 获取当前日期，确定复盘的交易日。

### 第二步：获取资金面核心数据

使用 `finx gildata-aidata` 查询以下数据：

1. **DR 系列利率**：DR001、DR007、DR014、DR021、DR1M
   ```
   调用 gildata-aidata MacroIndustryEDB，查询银行间回购利率：
   - 获取 DR 系列利率的最新数据
   - 支持时间序列查询，可获取日度、周度、月度数据
   ```

2. **同业存单利率**：1M、3M、6M、9M、1Y 同业存单发行利率/到期收益率
   ```
   调用 gildata-aidata NCDIssueRate，查询同业存单发行利率：
   - 获取按日统计的同业存单平均发行利率
   - 支持按期限（1M/3M/6M/9M/1Y）、主体评级、银行类别筛选
   - 获取发行规模及只数统计
   ```

3. **国债收益率曲线**：1Y、2Y、3Y、5Y、7Y、10Y、30Y 国债到期收益率
   ```
   调用 gildata-aidata MostActiveRateBondQuotation，查询利率债最活跃券行情：
   - 获取各期限最活跃国债的收益率及涨跌情况
   - 反映市场资金面风向和收益率曲线形态
   调用 gildata-aidata BondDailyClosingQuotes，查询债券日收盘行情：
   - 获取指定日期国债的收盘行情，包括到期收益率及久期
   ```

4. **SHIBOR 利率**：O/N、1W、2W、1M、3M
   ```
   调用 gildata-aidata MacroIndustryEDB，查询 SHIBOR 利率：
   - 获取 SHIBOR 各期限品种的报价数据
   - 支持时间序列查询
   ```

### 第三步：获取央行公开市场操作数据

使用 `web_search` 查询：
- 当日央行公开市场操作（逆回购、MLF、TMLF 等）
- 央行公告与政策动态
- 查询关键词示例："今日央行公开市场操作 逆回购 MLF"

### 第四步：获取宏观与市场舆情

使用 `finx gildata-aidata` 查询宏观舆情：
```
调用 gildata-aidata MacroNewslist，查询宏观舆情：
- 获取宏观经济预测及事件
- 支持按区域、情感类型筛选
调用 gildata-aidata PolicyMeetingsList，查询政策会议：
- 获取国内经济金融领域政策会议动态
```

使用 `web_search` 补充搜索：
- 当日资金面相关新闻与评论
- 流动性影响因素（缴税、政府债发行、跨月跨季等）
- 查询关键词示例："今日资金面 银行间流动性 央行投放"

## 数据处理与分析

### 利率变动计算

对获取的利率数据进行以下计算：

1. **日度变动**：当日利率 - 上一交易日利率（单位：BP）
2. **周度变动**：当日利率 - 上周同日利率（单位：BP）
3. **月度变动**：当日利率 - 上月同日利率（单位：BP）

### 期限利差分析

计算关键期限利差：

1. **国债期限利差**：
   - 10Y-1Y 利差
   - 10Y-2Y 利差
   - 30Y-10Y 利差

2. **同业存单期限利差**：
   - 1Y-3M 利差
   - 3M-1M 利差

3. **政策利率偏离度**：
   - DR007 相对 7 天逆回购利率的偏离
   - DR007 相对 MLF 利率的偏离

### 曲线形态判断

根据期限利差变化判断曲线形态：
- **陡峭化**：长端利率上行幅度 > 短端，或短端下行幅度 > 长端
- **平坦化**：长端利率下行幅度 > 短端，或短端上行幅度 > 长端
- **牛平/熊平/牛陡/熊陡**：结合利率方向与曲线形态综合判断

## 数据获取指引

### 使用 gildata-aidata 查询示例

**查询 DR 系列利率和 SHIBOR**：
```
finx gildata-aidata MacroIndustryEDB --body '{"indicator": "DR007,DR001,SHIBOR_O/N", "startDate": "2026-05-10", "endDate": "2026-05-11"}'
```

**查询同业存单发行利率**：
```
finx gildata-aidata NCDIssueRate --body '{"statType": "daily", "startDate": "2026-05-10", "endDate": "2026-05-11", "term": "1M,3M,6M,9M,1Y"}'
```

**查询利率债最活跃券行情**：
```
finx gildata-aidata MostActiveRateBondQuotation --body '{"tradeDate": "2026-05-11", "bondType": "government"}'
```

**查询债券日收盘行情**：
```
finx gildata-aidata BondDailyClosingQuotes --body '{"tradeDate": "2026-05-11", "bondType": "government"}'
```

**查询宏观舆情**：
```
finx gildata-aidata MacroNewslist --body '{"startDate": "2026-05-10", "endDate": "2026-05-11"}'
```

**查询政策会议**：
```
finx gildata-aidata PolicyMeetingsList --body '{"startDate": "2026-05-10", "endDate": "2026-05-11"}'
```

## 执行指引

1. **数据获取**：按照上述流程依次获取各类数据
2. **数据校验**：检查数据完整性，对缺失数据进行标注
3. **计算分析**：执行利率变动、期限利差等计算
4. **逻辑推演**：结合数据与市场信息，分析利率变动逻辑
5. **报告生成**：按照模板生成完整报告
6. **质量检查**：检查数据一致性、逻辑合理性

## 注意事项

- 利率变动单位统一使用 BP（1BP = 0.01%）
- 数据缺失时标注"数据暂缺"，不可编造数据
- 复盘结论需有数据支撑，避免主观臆断
- 交易建议仅供参考，不构成投资建议
- 报告生成后直接在对话中显示完整内容，无需发送文件
