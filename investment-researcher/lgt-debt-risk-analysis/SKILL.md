# 城投主体债务排雷分析技能

## 技能概述

本技能面向证券固定收益研究员，提供城投平台（地方政府融资平台）债务风险的自动化排雷分析能力。通过多维度数据交叉验证，识别潜在信用风险，生成结构化风险报告。

## 核心分析维度

### 1. 区域财政实力评估
- 一般公共预算收入及增速
- 政府性基金收入（土地出让金）
- 财政自给率（一般公共预算收入/一般公共预算支出）
- 上级转移支付依赖度

### 2. 债务压力分析
- 地方政府债务余额及债务率（债务余额/综合财力）
- 城投平台有息债务规模
- 宽口径债务率（地方政府债务+城投有息债务）/综合财力
- 债务期限结构与偿债高峰

### 3. 化债进度跟踪
- 特殊再融资债券发行情况
- 债务置换与展期进展
- 央行 SPV 工具使用情况
- 地方化债政策与具体措施

### 4. 主体信用排查
- 工商基本信息与股权穿透
- 司法诉讼与被执行人记录
- 行政处罚与经营异常
- 主体评级与评级展望
- 债券发行与兑付记录

## 执行流程

### 第一步：明确分析对象

从用户输入中提取：
- **城投主体名称**（如：XX 市城市建设投资集团）
- **所属区域**（省/市/区县）
- 如用户未指定具体主体，则按区域维度进行整体分析

### 第二步：获取区域财政与债务数据

使用 `finx gildata-aidata RegionalEconomicData` 查询目标区域的宏观经济与财政数据：

```
调用 gildata-aidata RegionalEconomicData，查询区域经济明细数据：
- 支持多级行政区划（省/市/区县）
- 获取 GDP、一般公共预算收入、政府性基金收入、财政收支、债务余额等核心指标
- 支持时间序列查询，可获取历年数据对比
```

关键数据点：
- 一般公共预算收入（绝对值及同比增速）
- 政府性基金收入（土地出让金占比）
- 一般公共预算支出
- 地方政府债务余额
- GDP 总量及增速

### 第三步：查询城投主体信息

使用 `finx gildata-aidata` 系列工具查询主体信用状况：

1. **区域城投平台明细数据**：
```
调用 gildata-aidata RegionalUrbanInvestmentData，查询区域城投平台明细数据：
- 获取城投平台风险评级、债务、财务等明细数据
- 支持按区域筛选，获取区域内所有城投平台概况
```

2. **发债主体基本信息**：
```
调用 gildata-aidata BasicInfoBondIssuer，查询发债主体基本资料：
- 获取注册信息、行业、存续债券等
- 了解主体基本资质
```

3. **工商基本信息**：
```
调用 gildata-aidata CompanyArchives，查询工商企业工商基本信息：
- 获取法人、注册资金、社保人数等
- 确认企业存续状态
```

4. **司法风险排查**：
```
调用 gildata-aidata Executee，查询企业被执行人信息：
- 获取立案时间、执行标的及案件状态

调用 gildata-aidata DishonestPersonList，查询企业失信被执行人信息：
- 获取立案时间、执行法院等

调用 gildata-aidata LimitHighConsume，查询企业限制高消费信息：
- 获取被限制高消费的执行法院等

调用 gildata-aidata CourtSession，查询企业开庭公告信息：
- 获取案号、案由、当事人、法院等
```

5. **经营合规排查**：
```
调用 gildata-aidata AbnormalOperationInfo，查询企业经营异常记录：
- 获取列入原因、移除情况等

调用 gildata-aidata RegulatoryPenaltyList，查询企业监管处罚信息：
- 获取处罚类型、原因、机构及金额

调用 gildata-aidata TaxArrearsAnnouncement，查询企业欠税公告：
- 获取税种、金额、税务机关等
```

6. **担保与质押风险**：
```
调用 gildata-aidata ExternalGuarantee，查询企业对外担保信息：
- 获取担保金额、对象及期限

调用 gildata-aidata CorporateEquityPledge，查询股东股权质押信息：
- 获取质押方、数量、日期等
```

7. **评级记录查询**：
```
调用 gildata-aidata BondRatingOverview，查询债券评级一览：
- 获取债项、发行人、担保人评级信息

调用 gildata-aidata BondIssuerRatingChange，查询发债主体评级变动：
- 获取主体评级上调/下调记录
- 支持按企业性质、评级机构筛选
```

### 第四步：获取财务数据

使用 `finx gildata-aidata` 查询主体财务状况：

```
调用 gildata-aidata FinancialStatements，获取发债公司历史财务报表：
- 覆盖三表主要科目
- 支持多报告期对比

调用 gildata-aidata BondFinancialAnalysis，获取发债公司财务分析指标：
- 盈利能力、偿债能力、营运能力、成长能力等
- 重点关注：资产负债率、流动比率、速动比率、利息保障倍数、经营性现金流等
```

### 第五步：搜索化债政策与舆情

使用 `finx gildata-aidata` 查询宏观舆情和政策会议：

```
调用 gildata-aidata MacroNewslist，查询宏观舆情：
- 获取宏观经济预测及事件
- 支持按区域、情感类型筛选

调用 gildata-aidata PolicyMeetingsList，查询政策会议：
- 获取国内经济金融领域政策会议动态
- 解析政策逻辑

调用 gildata-aidata NewsPublicOpinionList，检索全网新闻舆情：
- 搜索城投化债、债务置换等相关新闻
```

### 第六步：查询债券市场数据

使用 `finx gildata-aidata` 查询主体存续债券：

```
调用 gildata-aidata BasicInfoBondIssuer，查询发债主体基本信息：
- 获取存续债券列表

调用 gildata-aidata CreditBondBaseInfo，查询债券基本资料：
- 获取发行、评级、付息、行权等要素

调用 gildata-aidata DefaultBondList，查询违约债券清单：
- 确认主体是否存在违约记录

调用 gildata-aidata BondIssuerFirstDefault，查询发债主体首次违约：
- 获取违约日期、行业等
```

### 第七步：计算核心指标与风险评级

#### 债务率计算公式

| 指标 | 公式 | 风险阈值参考 |
|------|------|-------------|
| 财政自给率 | 一般公共预算收入 / 一般公共预算支出 × 100% | <50% 预警 |
| 狭义债务率 | 地方政府债务余额 / 综合财力 × 100% | >100% 预警 |
| 宽口径债务率 | (地方政府债务+城投有息债务) / 综合财力 × 100% | >300% 高危 |
| 土地依赖度 | 政府性基金收入 / 综合财力 × 100% | >40% 预警 |

> 注：综合财力 = 一般公共预算收入 + 政府性基金收入 + 上级转移支付

#### 风险等级标注体系

| 风险等级 | 标识 | 判定标准 |
|---------|------|---------|
| **安全** | 🟢 | 债务率<100%，财政自给率>60%，无负面舆情，评级稳定 |
| **关注** | 🟡 | 债务率100%-200%，财政自给率40%-60%，存在少量诉讼 |
| **预警** | 🟠 | 债务率200%-300%，财政自给率<40%，土地依赖度高，评级展望负面 |
| **高危** | 🔴 | 宽口径债务率>300%，存在失信被执行记录，债券展期或技术违约 |

## 数据处理原则

1. **数据时效性**：优先使用最新可得数据，标注数据截止日期
2. **交叉验证**：同一指标从多个数据源获取时，取最新或最权威来源
3. **缺失处理**：如某项数据无法获取，标注"数据暂缺"，不影响其他维度分析
4. **保守原则**：风险等级判定采取保守原则，宁可高估不可低估

## 注意事项

- 城投债务数据部分为非公开信息，宽口径债务率可能需要估算
- 化债政策变化较快，需关注最新政策动态
- 主体司法信息可能存在滞后，建议定期复查
- 风险等级仅为参考，实际投资决策需结合更多定性分析
