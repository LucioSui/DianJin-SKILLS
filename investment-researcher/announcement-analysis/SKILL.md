# 重大公告解读技能

## 技能定位

本技能帮助基金行业研究员、投资经理等专业人士快速、专业地解读上市公司重大公告，**以 gildata-aidata 金融数据服务为核心数据源**，生成结构化的解读报告。核心方法论采用券商研报的标准点评框架，注重边际变化提取和预期差分析。

**核心优势**：
- ✅ **gildata-aidata 专业数据源**：所有财务数据、估值数据、一致预期优先通过 gildata-aidata 服务获取
- ✅ **分析框架专业**：采用券商研报标准点评框架，注重边际变化和预期差分析
- ✅ **覆盖场景全面**：支持 13 种公告类型的专项分析模板
- ✅ **输出结构化**：标准化 Markdown 格式报告，适合投资决策参考

**输出格式**：
- 默认生成结构化 Markdown 格式报告
- 如用户明确要求 Word 文档，可调用相关工具生成
- 报告内容包含核心公告概览、财务数据分析、投资要点总结、风险提示等完整内容

## 工作流程

### Step 1：评估输入信息

**首先向用户确认以下信息：**

1. **公告来源**：用户是否提供了公告原文文件（PDF/文本）、公告链接，或仅提供了公司名称和公告类型？
2. **公司信息**：公司名称和股票代码（如用户未提供，需询问）
3. **公告类型**：属于哪一类（若用户未说明，根据公告内容自动识别）

**信息获取策略：**

- 若用户已提供公告文件/链接 → 直接进入 Step 2
- 若用户仅提供公司名称 + 公告类型 → **优先使用 gildata-aidata 服务查询公告信息**，辅以 `web_search` 搜索最新公告
- 若信息不足 → 向用户询问

### Step 2：公告分类与重要性评级

根据公告类型自动评级：

| 评级 | 公告类型 |
|------|----------|
| ★★★★★ | 年报、半年报、重大资产重组、控制权变更 |
| ★★★★☆ | 季报、业绩预告、定增、重大合同、大比例增减持（>5%） |
| ★★★☆☆ | 业绩快报、股权激励、回购、对外投资、小比例增减持 |
| ★★☆☆☆ | 关联交易、股权质押、董监高变动、一般诉讼 |

#### 数据获取执行顺序

根据公告类型选择对应的 gildata-aidata 工具调用顺序：

**定期报告/业绩类公告**：
1. `AShareFinancialReportReview`（仅 A 股）：获取现成的财报业绩点评文本，作为分析参考
2. `FinancialStatement`：获取三大报表核心数据
3. `FinancialAnalysis`：获取财务分析指标
4. `ConsensusExpectation`：获取市场一致预期，判断业绩是否超预期
5. `StockValueAnalysis`：获取估值数据

**资本运作类公告（定增/并购/配股）**：
1. `PrivatePlacement` 或相关工具：获取发行方案详情
2. `FinancialStatement`：获取公司财务基础数据
3. `StockValueAnalysis`：获取估值数据，判断发行价格合理性
4. `ConsensusExpectation`：获取一致预期，分析 EPS 摊薄影响

**股权变动类公告（增减持/回购/质押）**：
1. `ShareholdingChange` / `Repurchase` / `SharePledge`：获取变动详情
2. `StockValueAnalysis`：获取当前估值，判断变动价格合理性
3. `FinancialStatement`：获取财务数据，分析变动影响

**经营事项类公告（合同/投资/激励）**：
1. `MajorContract` / `EquityIncentive` 等：获取公告详情
2. `FinancialStatement`：获取财务基础数据，分析影响程度
3. `MainOperIncData`：获取业务结构，分析协同效应

**如 gildata-aidata 数据不完整**：使用 `web_search` 补充查询

### Step 4：核心分析（六要素框架）

对公告进行系统性分析，覆盖以下六大要素：

**1. 事件性质**：公告类型识别、触发原因分析、合规性判断、是否首次披露

**2. 核心内容**：提取关键数据（金额、比例、期限、条件）、核心条款梳理、交易对手/合作方信息

**3. 财务影响**（根据公告类型选择适用维度）：
- 定期报告：收入/利润增速、毛利率/净利率变化、现金流质量、ROE 杜邦分解
- 资本运作：EPS 摊薄/增厚、负债率变化、募资用途合理性、标的估值
- 经营事项：收入贡献比、利润率预估、产能利用率、执行周期
- 股权变动：变动金额、价格、比例、对股权结构的影响

**4. 战略意义**：对公司长期发展的影响、竞争格局变化、产业链位置变化

**5. 预期差分析**（核心分析）：
- 市场此前的一致预期是什么？
- 公告带来了哪些边际变化/新增信息？
- 哪些信息可能被市场误读或忽视？
- 结论：超预期 / 符合预期 / 低于预期

**6. 投资建议**：综合评级建议、核心看多/看空逻辑、需关注的后续催化剂

### Step 5：按公告类型进行专项分析

根据公告类型选择对应的专项分析模板（详见技能文件后半部分）。

### Step 6：生成结构化报告

**Step 6.1：组织内容结构**

按照下方【报告结构】组织报告内容，采用 Markdown 格式输出。

**Step 6.2：格式化输出**

1. 使用 Markdown 标题层级（#、##、###）组织章节
2. 使用表格展示财务数据和对比信息
3. 使用列表展示要点和风险提示
4. 使用粗体强调关键数据和结论
5. 确保逻辑清晰、层次分明

**Step 6.3：输出报告**

将完整的结构化报告直接呈现给用户，无需生成文件（除非用户特别要求）。

## 报告结构（Markdown 格式）

### 1. 主标题
`# [公司名称]（[股票代码]）重点公告解读`

### 2. 编制日期
`编制日期：YYYY 年 MM 月 DD 日`

### 3. 核心公告概览（多公告场景）
- 使用表格展示多个公告的对比
- 列：公告类型 | 披露日期 | 核心内容 | 关键数据
- 适用于同时解读多个关联公告的场景

### 4. 各公告专项解读（按公告类型分章节）

**业绩快报/财报类：**
- 主要财务数据表（7-8 行 5 列）：指标 | 本期 | 上期 | 同比变动 | 备注
- 财务状况表（4-5 行 4 列）：指标 | 期末值 | 期初值 | 变动
- 业绩亮点分析（分收入端、利润端、现金流等维度）

**担保/关联交易类：**
- 担保事项详情表
- 担保余额情况
- 风险分析（1-2 段）

**分红类：**
- 分红方案表（5 行 2 列）
- 分红政策分析（分红比例、频率、累计规模）
- 股息率测算

**回购类：**
- 回购详情表
- 回购影响分析（EPS、ROE、信心传递、资本结构）
- 与分红协同效应

### 5. 综合评估与投资建议
- 公司优势表（5 行 2 列）：维度 | 评估
- 需关注事项（编号列表，3-4 条）
- 投资要点总结（1-2 段）
- 核心亮点（使用 √ 符号标记，3-4 条）
- 风险提示（使用 △ 符号标记，2-3 条）

### 6. 重要时间节点
- 表格形式：时间 | 事项
- 5-6 行关键日期

### 7. 免责声明
> 免责声明：本解读基于公开信息整理，仅供参考，不构成任何投资建议。投资者应注意投资风险，独立判断，谨慎决策。

## 格式规范

### 标题层级
- **主标题**：使用 `#` 一级标题
- **一级标题**：使用 `##` 二级标题（如"一、核心公告概览"）
- **二级标题**：使用 `###` 三级标题（如"2.1 主要财务数据"）

### 表格
- 使用 Markdown 标准表格语法
- 表头与数据行之间使用 `|---|` 分隔
- 确保列对齐清晰

### 强调
- **粗体**：使用 `**文本**` 强调关键数据、结论
- *斜体*：使用 `*文本*` 表示说明性内容

### 列表
- 编号列表：使用 `1.` `2.` `3.` 展示有序内容
- 无序列表：使用 `-` 展示并列要点

### 特殊符号
- **核心亮点**：使用 √（Unicode: \u221A）
- **风险提示**：使用 △（Unicode: \u25B3）

## 专项分析模板

### 模板一：定期报告（年报/半年报/季报）

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 收入端 | 总营收增速（同比/环比）、分业务收入拆解、分地区收入结构、新业务占比变化 |
| 利润端 | 归母净利润增速、扣非净利润增速、毛利率/净利率变化趋势、费用率（销售/管理/研发/财务） |
| 现金流 | 经营性现金流/净利润比值（>1 为佳）、收现比、资本开支节奏 |
| 资产质量 | 存货变动及周转天数、应收账款变动及账龄、商誉占净资产比、有息负债率 |
| 股东结构 | 前十大股东变化、机构持仓变动、北向资金变动 |
| ROE 杜邦分解 | 净利率 × 总资产周转率 × 权益乘数，识别 ROE 变动的核心驱动因素 |

---

### 模板二：业绩预告/业绩快报

**核心分析维度：**

- **区间分析**：预告区间中值 vs 一致预期，判断超预期程度
- **单季拆解**：Q4 单季（或对应季度）业绩反推，与 Q3 环比对比
- **变动原因**：区分经常性损益和非经常性损益影响
- **全年节奏**：各季度利润占比是否符合历史季节性规律
- **指引信号**：管理层对下一年度的经营展望、资本开支计划

---

### 模板三：并购重组

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 交易方案 | 交易对价总额、支付方式（现金/股份/混合）、锁定期安排 |
| 标的质量 | 标的公司营收/利润规模及增速、所处行业地位、核心竞争力 |
| 估值合理性 | PE/PB/PS 倍数 vs 同行业可比交易、溢价率（vs 账面净资产） |
| 业绩承诺 | 承诺期限、承诺利润金额、年均复合增速要求、补偿条款 |
| 协同效应 | 业务协同（客户/渠道/产品互补）、成本协同、财务协同 |
| EPS 影响 | 测算交易完成后对每股收益的摊薄/增厚效应 |

---

### 模板四：定增/配股/可转债

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 发行方案 | 拟募资金额、发行数量上限、定价方式（竞价/锁价）、锁定期 |
| 募投项目 | 各项目投资金额、建设周期、预计达产时间、预期收益率 |
| 股本摊薄 | 增发股数/现有总股本比例、对 EPS 的摊薄程度 |
| 认购方 | 大股东/战略投资者认购比例（高比例认购视为利好信号） |
| 资金用途合理性 | 与公司现有业务的协同性、是否解决产能瓶颈 |

---

### 模板五：增减持公告

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 变动主体 | 控股股东/实控人、高管、机构投资者（不同主体信号意义不同） |
| 变动规模 | 变动股份数、占总股本比例、涉及金额 |
| 变动价格 | 交易均价 vs 当前股价、vs 历史买入成本 |
| 变动原因 | 自身资金需求、看好公司发展（增持）、解禁减持、基金到期（减持） |
| 后续计划 | 是否有继续增减持计划、增持承诺（金额/期限） |

**信号解读：**
- 增持：控股股东/高管增持通常为积极信号（尤其是大额、自有资金增持）
- 减持：需区分被动减持（解禁、质押平仓）和主动减持，主动大额减持偏负面

---

### 模板六：股权激励/员工持股

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 激励范围 | 覆盖人数、核心技术/管理人员占比 |
| 授予价格 | 行权价/授予价 vs 当前股价的折价率 |
| 考核指标 | 收入增速目标、利润增速目标、ROE 等回报指标要求 |
| 考核挑战性 | 对比历史增速和行业水平，判断目标是否具有挑战性 |
| 费用影响 | 股份支付费用总额、摊销期限、各年度对利润的影响比例 |

---

### 模板七：重大合同

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 合同规模 | 合同金额、占最近一期年度营收的比例（>10% 视为重大） |
| 合同对手 | 客户行业地位、信用状况、历史合作情况 |
| 执行节奏 | 合同期限、分年度/分阶段执行安排、验收节点 |
| 收入确认 | 按完工百分比法 or 终验法确认收入、对当年及未来各年度的业绩贡献 |
| 战略价值 | 是否打开新客户/新市场/新领域、是否具有标杆效应和可复制性 |

---

### 模板八：回购公告

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 回购方案 | 回购金额上下限、回购价格上限、回购期限、资金来源 |
| 回购目的 | 注销减少注册资本（最利好）、员工持股/股权激励、市值管理 |
| 回购力度 | 回购金额占市值比例、占自由现金流比例 |
| 信号意义 | 管理层对股价低估的判断、历史回购后股价表现 |
| 执行进度 | 已回购比例、剩余额度、回购节奏是否加速 |

---

### 模板九：股权质押/解押

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 质押规模 | 累计质押比例（>50% 需警惕）、本次质押占持股比例 |
| 质押目的 | 融资用途（个人需求 vs 公司经营 vs 投资） |
| 平仓风险 | 预估平仓线 vs 当前股价的安全垫、补充质押能力 |
| 解押信号 | 解押通常为积极信号，表明资金压力缓解 |

---

### 模板十：对外投资/关联交易

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 投资规模 | 投资金额占净资产比例、资金来源 |
| 投资标的 | 与主业的协同性、标的所处行业前景、估值合理性 |
| 关联交易公允性 | 定价依据、是否有独立评估、是否损害中小股东利益 |
| 审批程序 | 是否需要股东大会审议、关联股东是否回避表决 |

---

### 模板十一：风险事件（诉讼/处罚/退市风险）

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 事件性质 | 涉及金额、违规类型、严重程度 |
| 财务影响 | 预计赔偿/罚款金额 vs 公司净资产和净利润规模 |
| 经营影响 | 是否影响正常生产经营、是否导致资质丧失 |
| 声誉影响 | 对客户关系、融资能力、市场形象的影响 |
| 后续演进 | 案件进展时间表、是否有和解可能、最坏情景分析 |

---

### 模板十二：利润分配/分红

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 分红方案 | 每 10 股派息金额（含税）、送转股比例、分红总额 |
| 分红比例 | 占归母净利润比例、与历史分红比例对比 |
| 分红频率 | 年度分红、中期分红、特别分红 |
| 股息率 | 按当前股价测算股息率，与行业/债券收益率对比 |
| 累计分红 | 上市以来累计分红金额、分红年数 |
| 回购协同 | 分红 + 回购的总股东回报规模 |

---

### 模板十三：担保公告

**核心分析维度：**

| 维度 | 分析要点 |
|------|----------|
| 担保详情 | 担保方式、金额、期限、担保对象 |
| 担保余额 | 累计担保余额、占净资产比例 |
| 担保风险 | 是否对合并报表内子公司担保、有无逾期记录 |
| 额度来源 | 是否经股东大会批准、剩余额度 |
| 风险分析 | 担保比例是否低于警戒线（通常<30% 为安全） |

## 分析质量要求

1. **数据驱动**：所有结论必须有公告中的具体数据支撑，避免空泛描述
2. **边际思维**：重点分析"变化了什么"而非"是什么"
3. **预期对比**：务必将公告信息与市场一致预期进行对比
4. **正反兼顾**：既分析利好因素也指出风险隐患，保持客观中立
5. **简洁专业**：使用专业但不晦涩的语言，投资要点控制在 3-5 条
6. **格式规范**：严格遵循 Markdown 格式规范，确保输出结构清晰、层次分明

## 工具使用指引

| 工具 | 用途 |
|------|------|
| **finx gildata-aidata** | **核心数据源**，获取财务报表、财务分析指标、主营构成、一致预期、估值、定增、回购、增减持、股权激励、重大合同、股权质押、诉讼处罚、对外担保等数据 |
| `web_search` | 补充查询，用于 gildata-aidata 无法覆盖的公告原文、舆情、市场评论等 |
| `web_fetch` | 从具体 URL 获取公告原文、研报详细内容 |
| `file` | 读取用户上传的公告文件（PDF/文本） |

**重要原则**：
- **所有财务数据、估值数据、一致预期查询必须优先使用 gildata-aidata 服务**
- 如 gildata-aidata 数据不完整可辅以 web_search 查询
- 如用户提供公告文件，优先解析文件数据，并用 gildata-aidata 补充衍生数据
- 所有数据获取后需进行交叉验证，确保数据准确性

## 注意事项

1. 若用户未提供公告原文，提示用户上传或提供链接
2. 专业术语首次出现时简要解释
3. 报告末尾必须添加免责声明
4. 解读日期使用当前日期
5. **数据来源优先级：gildata-aidata > 公司公告 > 券商研报 > 财经新闻**
6. 多公告场景下，使用表格对比展示各公告核心信息
7. 使用 √ 和 △ 符号标记亮点和风险，增强可读性
8. 如用户明确要求 Word 文档，可告知用户当前默认输出为 Markdown 格式，如需 Word 可另行处理
9. **数据缺失处理**：如 gildata-aidata 部分数据无法获取，在对应位置标注"未披露"，不可编造数据

## gildata-aidata 工具调用指南

### 查询格式说明

gildata-aidata 工具使用自然语言 `query` 参数，调用格式如下：

```bash
finx gildata-aidata <工具名> --body '{"query": "<公司名称> <报告期> <查询内容>"}'
```

### A 股公司常用查询模板

```bash
# 财务报表（三大表）
finx gildata-aidata FinancialStatement --body '{"query": "五粮液 2024 年年报 利润表 资产负债表 现金流量表"}'

# 财务分析指标
finx gildata-aidata FinancialAnalysis --body '{"query": "五粮液 2024 年 毛利率 净利率 ROE ROIC 资产负债率 应收账款周转率 存货周转率"}'

# 主营构成
finx gildata-aidata MainOperIncData --body '{"query": "五粮液 2024 年 分产品 分行业 分地区 收入 成本 毛利率"}'

finx gildata-aidata ConsensusExpectation --body '{"query": "五粮液 2024 2025 2026 营收 净利润 EPS 一致预期"}'

# 估值数据
finx gildata-aidata StockValueAnalysis --body '{"query": "五粮液 PE PB PS EV 股息率"}'

# 机构评级
finx gildata-aidata InstitutionalRating --body '{"query": "五粮液 机构评级 评级分数"}'

# 业绩预告

# 业绩快报
```
