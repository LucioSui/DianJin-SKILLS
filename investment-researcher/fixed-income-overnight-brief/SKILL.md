# 固定收益隔夜舆情速览

## 技能定位

本技能专为证券固定收益研究员设计，用于快速聚合隔夜及最新债券市场相关舆情，生成每日雷点清单和速览报告，帮助研究员在开盘前掌握市场动态、识别信用风险、跟踪政策变化。

## 触发场景

- 用户要求生成隔夜固收舆情速览、固收晨报、债市日报
- 用户询问今日债市有什么风险点、雷点、需要关注的事件
- 用户要求跟踪债券违约、评级下调、评级展望负面等信用事件
- 用户要求汇总央行公开市场操作、MLF、LPR、降准降息等货币政策动态
- 用户要求梳理最新宏观政策、财政政策、监管政策对债市的影响
- 用户提及"固收舆情"、"债券市场要闻"、"信用债跟踪"、"利率债动态"等关键词

## 执行流程

### 第一步：获取当前日期和时间

调用 `finx time` 工具获取当前准确日期，用于确定报告覆盖的时间范围（通常为隔夜及最近一个交易日）。

### 第二步：多维度信息采集

并行执行以下查询，获取最新固收市场信息：

#### 1. 违约事件跟踪

使用 `finx gildata-aidata` 查询违约债券信息：

```
调用 gildata-aidata DefaultBondList，查询违约债券清单：
- 支持按时间、发行人、类型等多维筛选
- 获取违约日期、违约类型、涉及金额等核心信息

调用 gildata-aidata BondIssuerFirstDefault，查询发债主体首次违约：
- 获取违约日期、行业、企业性质等
- 识别新增违约主体
```

#### 2. 评级调整跟踪

使用 `finx gildata-aidata` 查询评级变动信息：

```
调用 gildata-aidata BondRatingChange，查询债项评级变动：
- 获取债项评级上调/下调记录
- 支持按评级机构、变动方向筛选

调用 gildata-aidata BondIssuerRatingChange，查询发债主体评级变动：
- 获取主体评级上调/下调记录
- 支持按企业性质、评级机构筛选

调用 gildata-aidata GuarantorRatingChange，查询担保人评级变动：
- 获取担保人评级变动记录

调用 gildata-aidata CCDCImpliedRatingChange，查询中债隐含评级变动：
- 获取隐含评级变动，反映市场真实信用预期
```

#### 3. 央行操作与资金面

使用 `finx gildata-aidata` 查询资金利率数据：

```
调用 gildata-aidata NCDIssueRate，查询同业存单发行利率：
- 获取按日统计的同业存单平均发行利率
- 支持按期限、主体评级、银行类别筛选

调用 gildata-aidata MostActiveRateBondQuotation，查询利率债最活跃券行情：
- 获取各期限最活跃券的收益率及涨跌情况
- 反映市场资金面风向

使用 web_search 补充查询央行公开市场操作：
- "央行公开市场操作 最新"
- "MLF 操作 最新"
- "LPR 报价 最新"
- "逆回购 投放 回笼"
- "DR007 SHIBOR 最新"
```

#### 4. 宏观政策与监管动态

使用 `finx gildata-aidata` 查询宏观舆情和政策会议：

```
调用 gildata-aidata MacroNewslist，查询宏观舆情：
- 获取宏观经济预测及事件
- 支持按区域、情感类型筛选

调用 gildata-aidata PolicyMeetingsList，查询政策会议：
- 获取国内经济金融领域政策会议动态
- 解析政策逻辑

调用 gildata-aidata BondIssueMatNetFin，查询债券发行与到期统计：
- 获取按周/月统计债券总发行、总偿还及净融资额
- 细分到期、提前兑付、回售、赎回等偿还结构

调用 gildata-aidata MarketIssueStatByBondType，查询债券发行统计：
- 统计债券发行量，支持按类型、期限、评级筛选
```

#### 5. 债市行情与热点

使用 `finx gildata-aidata` 查询债市行情和舆情：

```
调用 gildata-aidata BondDailyClosingQuotes，查询债券日收盘行情：
- 获取债券在指定日期的收盘行情
- 包括价量、到期收益率及久期

调用 gildata-aidata BondRiskReturnAnalysis，查询债券风险收益分析：
- 获取债券的收益率、久期、凸性、信用利差等风险收益指标

调用 gildata-aidata SecurityNewslist，查询债券舆情：
- 获取债券相关舆情
- 支持按情感类型、证券代码筛选风险信息

调用 gildata-aidata InterestRateDebtViewpoints，查询利率债观点：
- 获取研报中对利率债基本面、政策、资金面等的分析观点
```

### 第三步：信息筛选与分类

对查询结果进行筛选和分类：

1. **剔除无关信息**：过滤与固收市场无关的结果
2. **去重合并**：合并同一事件的多个报道
3. **重要性排序**：按事件影响力排序（重大违约 > 评级下调 > 政策变化 > 常规操作）
4. **时效性确认**：确保信息为最新（24-48 小时内）

按照以下模板生成 Markdown 格式报告：

## 注意事项

1. **数据真实性**：所有数据必须来源于查询结果，不可编造。如某项数据无法获取，标注"待更新"或"暂无数据"
2. **时效性**：优先使用 24-48 小时内的最新信息，标注信息日期
3. **客观中立**：仅陈述事实，主观判断需在"研究员关注提示"中明确标注为个人观点
4. **风险优先**：雷点清单应放在报告前部，便于快速阅读
5. **格式规范**：表格数据对齐，缺失数据用"-"填充，不使用"未知"等模糊表述
6. **引用标注**：重要事件应标注信息来源链接

## 输出要求

- 报告生成后直接在对话中显示完整 Markdown 内容
- 无需生成文件，除非用户明确要求保存
- 如用户要求定时生成，可建议创建定时任务
