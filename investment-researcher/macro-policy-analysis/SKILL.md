# 宏观政策跟踪与推演分析技能

## 概述

本技能用于自动抓取全球及中国核心政策动态，进行政策意图解读、预期推演和落地效果跟踪，生成面向投资专业人士的政策分析简报。

## 数据获取策略

### 核心数据源

**Gildata-AIData (gildata-aidata)** 作为主要数据源，提供全方位的政策、会议、讲话及舆情数据：

| 数据需求 | 推荐工具 | 说明 |
|---------|---------|------|
| 政策会议 | `PolicyMeetingsList` | 国内经济金融领域政策会议动态及内容 |
| 官员讲话 | `OfficialSpeechEventList` | 国内外重要官员讲话及活动信息 |
| 法律法规 | `LawsRegulations` | 法律法规信息，按类型、关键词筛选 |
| 宏观舆情 | `MacroNewslist` | 宏观舆情，按区域、情感类型筛选经济预测及事件 |
| 市场舆情 | `MarketNewslist` | 市场舆情，按市场类型筛选 |
| 全网舆情 | `NewsInfoList` | 全网舆情，按关键词、来源、情感类型筛选 |
| 宏观数据验证 | `MacroIndustryEDB` | 宏观行业经济数据，用于验证政策效果（如GDP、CPI、PMI等） |
| 宏观分析观点 | `MacroeconomicAnalysisViewpoints` | 研报中对GDP、CPI、PMI、政策等宏观维度的分析观点 |

### 数据获取优先级

1. **政策会议与法规**：优先 `PolicyMeetingsList` + `LawsRegulations` 获取最新政策动态
2. **官员讲话与定调**：优先 `OfficialSpeechEventList` 获取央行、财政部、发改委等官员讲话
3. **宏观舆情与观点**：优先 `MacroNewslist` + `MacroeconomicAnalysisViewpoints` 获取市场解读
4. **网络搜索作补充**：对于海外政策（美联储等）细节、最新突发新闻，使用 `web_search` 补充

## 执行流程

### 第一阶段：政策信息抓取

1. **全球主要经济体政策动态**

   **使用 gildata-aidata 查询**：
   ```
   调用 gildata-aidata OfficialSpeechEventList：
   查询海外重要官员讲话（如美联储主席讲话等）
   
   调用 gildata-aidata MacroNewslist：
   query: "美联储 欧央行 海外通胀 大宗商品"
   ```
   
   **使用 web_search 补充（海外政策）**：
   - 搜索关键词示例："美联储 最新政策 利率决议"、"欧央行 货币政策"、"美债收益率 最新走势"、"海外通胀数据 CPI PCE"、"国际大宗商品 原油 黄金 铜价"
   - 重点关注：美联储利率决议与点阵图、欧央行/日央行/英央行货币政策、美国财政与债务动态、全球主要经济体通胀与就业数据

2. **国内政策动态**

   **使用 gildata-aidata 查询**：
   ```
   调用 gildata-aidata PolicyMeetingsList：
   查询最新政策会议动态
   
   调用 gildata-aidata LawsRegulations：
   query: "央行 货币政策 降准 降息 专项债 减税降费"
   
   调用 gildata-aidata MacroNewslist：
   query: "央行 货币政策 OMO MLF LPR 财政政策 专项债"
   ```
   
   重点关注：央行公开市场操作、MLF/LPR报价、降准降息预期、财政预算与专项债发行、房地产政策、资本市场改革、行业监管政策

3. **核心宏观舆情**

   **使用 gildata-aidata 查询**：
   ```
   调用 gildata-aidata OfficialSpeechEventList：
   查询央行行长、发改委官员等讲话
   
   调用 gildata-aidata MacroNewslist：
   query: "政治局会议 经济 国务院常务会议 发改委 产业政策"
   
   调用 gildata-aidata MacroeconomicAnalysisViewpoints：
   query: "货币政策 财政政策 最新分析观点"
   ```
   
   重点关注：高层经济会议定调、央行/财政部/发改委官员讲话、重要行业政策导向（新能源、半导体、房地产、平台经济等）

### 第二阶段：政策分类与深度解读

4. **政策分类整理**

   | 政策类别 | 关注要点 |
   |---------|---------|
   | 货币政策 | 利率调整、准备金率、公开市场操作、结构性工具、信贷窗口指导 |
   | 财政政策 | 财政赤字、专项债发行、减税降费、转移支付、基建投资 |
   | 产业政策 | 行业扶持/限制、技术标准、产能调控、进出口政策 |
   | 监管政策 | 金融监管、资本市场改革、房地产调控、平台经济规范 |
   | 海外政策 | 美联储/欧央行利率决议、海外财政刺激、贸易政策、地缘政治 |

5. **政策意图解读**
   - 分析政策出台的背景与动因（经济数据压力、外部环境变化、结构性问题等）
   - 解读政策信号（宽松/中性/收紧、稳增长/防风险、支持/限制某行业）
   - 识别政策基调变化（相比前期政策的边际变化、措辞微调背后的含义）

6. **实施路径分析**
   - 政策工具选择及其传导机制（如降准→银行流动性→信贷投放→实体经济）
   - 政策力度评估（超预期/符合预期/低于预期）
   - 政策时滞判断（立竿见影型、中期见效型、长期影响型）

### 第三阶段：政策预期推演

7. **后续政策推演**

   **货币政策推演维度：**
   - 结合当前经济数据（GDP、CPI、PPI、社融、就业）判断政策空间
   - 推演未来1-3个月降准/降息/结构性工具出台的概率与节奏
   - 分析央行货币政策执行报告、官员讲话中的前瞻性指引

   **财政政策推演维度：**
   - 财政赤字率与专项债额度使用进度
   - 特别国债、地方化债政策后续安排
   - 财政发力节奏（前置/后置）及对基建投资的拉动

   **产业与监管政策推演维度：**
   - 重要会议（两会、政治局会议、中央经济工作会议）定调方向
   - 行业政策周期判断（扶持期、规范期、调整期）
   - 监管态度边际变化（鼓励创新、防范风险、规范发展）

8. **大类资产配置影响预判**

   | 资产类别 | 宽松政策影响 | 收紧政策影响 |
   |---------|-------------|-------------|
   | 利率债 | 利多（收益率下行） | 利空（收益率上行） |
   | 信用债 | 利差收窄 | 利差走阔 |
   | 股市（整体） | 流动性支撑、估值提升 | 流动性收紧、估值承压 |
   | 股市（板块） | 金融地产、成长股受益 | 银行、价值股相对抗跌 |
   | 大宗商品 | 需求预期改善、美元走弱支撑 | 需求预期走弱、美元走强压制 |
   | 汇率 | 本币短期承压 | 本币短期走强 |

### 第四阶段：政策落地跟踪

9. **前期政策效果评估**

   **使用 gildata-aidata 验证数据**：
   ```
   调用 gildata-aidata MacroIndustryEDB：
   query: "信贷 投资 消费 就业 最新数据"
   ```
   
   - 跟踪过去1-3个月出台的重大政策落地情况
   - 分析政策传导过程中的堵点（如宽货币未宽信用、财政支出慢、企业投资意愿不足等）
   - 评估政策对经济的实际拉动作用（数据验证：信贷、投资、消费、就业等）

10. **政策预判调整**
    - 根据落地效果修正前期政策预期
    - 识别政策"雷声大雨点小"或"超预期发力"的情形
    - 提出后续政策优化建议或加码/退出预判

## 注意事项

- **政策时效性**：优先使用当日或最近一周发布的政策信息，注明政策发布日期
- **解读客观性**：政策解读应基于官方文本和权威渠道，避免主观臆测，区分事实与观点
- **推演逻辑性**：政策推演需建立在经济数据和政策框架基础上，给出概率判断而非确定性结论
- **跟踪连续性**：政策落地跟踪应与前期报告形成呼应，体现政策效果的动态演变
- **海外联动性**：关注海外政策对国内市场的传导路径（如美联储加息对人民币汇率、资本流动的影响）
- **gildata-aidata 优先**：所有国内政策、会议、讲话、舆情查询优先使用 gildata-aidata 服务，web_search 仅用于补充海外政策细节

## 输出要求

- 报告使用标准Markdown格式
- 政策引用需注明出处（如"央行2026年4月MLF操作"）
- 推演结论应给出概率判断（如"降准概率60%"）而非绝对断言
- 报告长度控制在2000-3000字为宜，聚焦核心政策信号与投资启示

## 示例调用

用户输入："今日政策解读"

技能执行：
1. 获取政策会议：gildata-aidata PolicyMeetingsList
2. 获取官员讲话：gildata-aidata OfficialSpeechEventList
3. 获取法律法规：gildata-aidata LawsRegulations
4. 获取宏观舆情：gildata-aidata MacroNewslist + MacroeconomicAnalysisViewpoints
5. 补充海外政策：web_search 查询美联储等最新动态
6. 验证政策效果：gildata-aidata MacroIndustryEDB 查询最新宏观数据
7. 撰写报告：按模板生成 Markdown 格式政策推演分析报告
8. 输出报告：直接在对话中显示完整报告内容
