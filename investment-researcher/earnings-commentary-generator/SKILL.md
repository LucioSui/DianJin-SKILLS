# 上市公司财报业绩点评报告生成技能

## 技能概述

本技能专为上市公司财报（年报/半年报/季报）分析设计，**以 gildata-aidata 金融数据服务为核心数据源**，生成结构化业绩点评报告。报告包含业绩概览、核心财务指标分析、业务板块拆解、盈利质量评估、风险提示及投资观点参考，帮助投资者快速把握公司经营状况。

**核心优势**：
- ✅ **gildata-aidata 专业数据源**：所有财务数据优先通过 gildata-aidata 服务获取，确保数据权威准确
- ✅ **数据全面**：覆盖营收、利润、毛利率、现金流、ROE 等核心指标
- ✅ **分析专业**：采用买方研究员分析框架，识别经营亮点与风险点
- ✅ **报告结构化**：8 大模块标准化输出，适合投资决策参考

---

## 触发场景

当用户提及以下任一表述时触发本技能：
- "XX 公司业绩点评"、"XX 公司财报分析"、"XX 公司业绩分析"
- "分析 XX 公司财报"、"点评 XX 公司业绩"、"生成财报点评报告"
- "XX 公司年报点评"、"XX 公司季报点评"、"XX 公司半年报分析"
- "业绩快评"、"财报解读"、"业绩点评"、"季报点评"、"财报分析报告"、"买方点评"、"盈利预测调整"
- 上传财报 PDF 并要求分析

**即使用户只说"分析 XX 公司业绩"、"点评 XX 公司财报"等简短表述也要触发。**

## 工作流程

### 第一步：识别公司与报告期

1. **从用户输入提取公司信息**：
   - 公司名称（如"五粮液"、"贵州茅台"）
   - 股票代码（如有提供，如"000858"、"600519"）
   - 报告期（如"2024 年报"、"2025 一季报"，如未指定则获取最新报告期）
   - 是否有上传 PDF 文件
   - 市场类型（A 股/港股/美股，如未明确则默认 A 股）

2. **确定数据获取方式**：
   - **有 PDF**：解析 PDF 提取数据，并用 gildata-aidata 补充查询估值、一致预期等衍生数据
   - **无 PDF**：完全通过 gildata-aidata 服务获取全部财务数据

3. **确定财报类型**：
   - 年报（年度报告）
   - 半年报（中期报告）
   - 季报（一季度/三季度报告）

#### 数据获取执行顺序

严格按照以下顺序调用 gildata-aidata 工具：

1. **`AShareFinancialReportReview`**（仅 A 股）：获取现成的财报业绩点评文本，作为分析参考
2. **`FinancialStatement`**：获取三大报表核心数据（营收、利润、现金流、资产负债）
3. **`FinancialAnalysis`**：获取财务分析指标（毛利率、净利率、ROE、周转率等）
4. **`MainOperIncData`**：获取主营构成数据（分产品/分行业/分地区）
5. **`ConsensusExpectation`**：获取市场一致预期，判断业绩是否超预期
6. **`StockValueAnalysis`**：获取估值数据（PE、PB 等）
7. **`InstitutionalRating`**：获取机构评级情况
8. **`BonusStock`**：获取分红信息（年报时）
9. **如 gildata-aidata 数据不完整**：使用 `web_search` 补充查询

| 指标类别 | 具体指标 | gildata-aidata 数据源 |
|---------|---------|---------------------|
| 营收利润 | 营业收入、归母净利润、扣非净利润 | `FinancialStatement` |
| 盈利能力 | 毛利率、净利率、ROE、ROIC | `FinancialAnalysis` |
| 营运能力 | 应收账款周转率、存货周转率 | `FinancialAnalysis` |
| 偿债能力 | 资产负债率、流动比率、速动比率 | `FinancialAnalysis` |
| 现金流 | 经营性现金流、自由现金流、资本开支 | `FinancialStatement` |
| 成长能力 | 营收增速、利润增速、三年复合增速 | 基于 `FinancialStatement` 计算 |

### 第三步：整理关键财务指标

从 gildata-aidata 获取的数据中提取并整理以下财务数据：

| 指标类别 | 具体指标 | 分析维度 |
|---------|---------|---------|
| 营收利润 | 营业收入、归母净利润、扣非净利润 | 同比/环比/ vs 预期 |
| 盈利能力 | 毛利率、净利率、ROE、ROIC | 变化趋势 |
| 营运能力 | 应收账款周转率、存货周转率、总资产周转率 | 效率变化 |
| 偿债能力 | 资产负债率、流动比率、速动比率 | 财务安全 |
| 现金流 | 经营性现金流、自由现金流、资本开支 | 现金质量 |
| 成长能力 | 营收增速、利润增速、三年复合增速 | 成长性评估 |

**数据要求**：
- 当期数据 + 上年同期数据（计算同比）
- 如有上期期末数据（计算环比，适用于季报）
- 标注超预期/低于预期的指标（如有市场一致预期）
- **如 gildata-aidata 部分数据无法获取，在对应位置标注"未披露"，并在报告中说明数据来源**

### 第四步：分析业绩变化驱动因素

基于 gildata-aidata 获取的财务数据和管理层讨论信息：

1. **收入增长驱动**：
   - 销量变化 vs 价格变化
   - 各业务板块贡献
   - 新业务/新产品贡献

2. **利润变化驱动**：
   - 毛利率变化原因（原材料、产品结构、规模效应等）
   - 费用率变化（销售/管理/研发/财务费用）
   - 非经常性损益影响

3. **现金流变化**：
   - 经营性现金流与净利润匹配度
   - 营运资本变动影响
   - 资本开支方向

### 第五步：识别经营亮点与风险点

**亮点识别**（3-5 条）：
- 超预期的财务指标
- 新业务/新市场突破
- 重要客户/订单进展
- 产能扩张/投产进度
- 技术创新/产品升级
- 成本费用优化

**风险识别**（3-5 条）：
- 业绩不及预期的指标
- 毛利率/净利率下滑
- 应收账款/存货大幅增长
- 经营性现金流恶化
- 重大投资/并购风险
- 行业政策变化影响
- 大客户依赖/集中度高

### 第六步：盈利质量与财务健康度评估

综合评估以下维度：

| 评估维度 | 评估指标 | 评判标准 |
|---------|---------|---------|
| 盈利真实性 | 扣非净利润/归母净利润 | >80% 为优 |
| 现金含量 | 经营现金流/净利润 | >1 为优 |
| 资产质量 | 应收账款增速 vs 营收增速 | 应收增速<营收增速为优 |
| 库存健康 | 存货增速 vs 营收增速 | 存货增速<营收增速为优 |
| 财务安全 | 资产负债率、有息负债率 | 负债率<60% 为安全 |
| 持续分红 | 分红率、股息率 | 稳定分红为加分项 |

## 特殊场景处理

### 场景 1：gildata-aidata 部分数据无法获取
- 在对应章节标注"未披露"或"数据暂无"
- 可通过 `web_search` 补充查询公开财务数据
- 在文档末尾说明数据来源和局限性

### 场景 2：不同财报类型适配
- **年报**：包含完整财务数据、审计意见、分红方案、业绩说明会信息
- **半年报**：数据完整度较高，可能无审计意见
- **季报**：数据相对简化，重点分析环比变化

### 场景 3：多期财报对比
如用户要求对比多期财报：
- 生成趋势分析图表数据
- 计算复合增速（CAGR）
- 识别长期趋势变化

### 场景 4：用户指定报告期
如用户指定"分析 XX 公司 2023 年年报"：
- 使用 gildata-aidata 查询对应报告期数据
- 如数据较旧，在报告中提示"财报时效性注意"

### 场景 5：港股/美股公司
- **港股**：使用 `HKFinancialStatement`、`HKMainOperIncData`、`HKStockValueAnalysis` 等 gildata-aidata 工具
- **美股**：使用 `USFinancialStatement`、`USFinancialAnalysis`、`USValuationMetrics` 等 gildata-aidata 工具
- 注意财报披露格式差异（如港股用"股东应占溢利"而非"归母净利润"）

---

## 输出后处理

**重要**：报告生成后直接在对话中输出完整的 Markdown 报告内容，让用户可以立即阅读。

**执行方式**：
- 直接将完整的 Markdown 报告内容在对话中输出（使用 Markdown 格式渲染）
- **不需要保存文件**
- **不需要发送文件给用户**

**注意**：
- 报告生成后，可以在对话中提供核心结论摘要，帮助用户快速把握要点
- 如报告过长（超过 15000 字），可在对话中先输出执行摘要和核心章节，再输出完整报告

## 使用示例

### 示例 1：基础业绩点评（无 PDF）

**用户输入**："五粮液业绩点评"

**技能执行**：
1. 识别公司：五粮液（000858.SZ）
2. 确定报告期：最新报告期（2024 年年报）
3. **使用 gildata-aidata 获取数据**：
   - `AShareFinancialReportReview`：查询"五粮液 2024 年年报 业绩点评"
   - `FinancialStatement`：查询"五粮液 2024 年年报 利润表 现金流量表"
   - `FinancialAnalysis`：查询"五粮液 2024 年 毛利率 净利率 ROE"
   - `MainOperIncData`：查询"五粮液 2024 年 分产品 收入 毛利率"
   - `ConsensusExpectation`：查询"五粮液 2024 2025 一致预期"
   - `StockValueAnalysis`：查询"五粮液 PE PB 股息率"
4. 整理财务指标，计算同比变化
5. 分析经营亮点与风险点
6. 生成业绩点评报告
7. 直接在对话中输出完整报告

**输出**：包含 8 大模块的完整业绩点评报告

### 示例 2：指定报告期

**用户输入**："分析贵州茅台 2025 年一季报"

## 注意事项

1. **gildata-aidata 优先**：所有财务数据必须优先通过 gildata-aidata 服务获取，不得臆造或估算
2. **数据准确性**：如 gildata-aidata 无法获取某项数据，可辅以 web_search 查询权威来源（公司公告、东方财富等）
3. **客观中立**：点评应基于事实数据，避免主观臆断
4. **风险提示**：风险部分不可省略，必须清晰呈现潜在风险
5. **口径一致**：同比/环比计算口径需一致，注明是否扣除并购影响
6. **时效性**：如财报发布超过 3 个月，在文档中提示"财报时效性注意"
7. **数据来源**：在报告末尾明确列出数据来源，便于用户核查
8. **数据缺失处理**：如 gildata-aidata 部分数据无法获取，在对应位置标注"未披露"，不可编造数据
9. **港股/美股适配**：对于港股/美股公司，注意财报披露格式差异（如港股用"股东应占溢利"而非"归母净利润"）

## 工具使用说明

本技能主要使用以下工具获取数据：

1. **finx gildata-aidata**：**核心数据源**，获取财务报表、财务分析指标、主营构成、一致预期、估值等数据
2. **web_search**：补充查询，用于 gildata-aidata 无法覆盖的舆情、新闻、业绩说明会详情等
3. **web_fetch**：从具体 URL 获取详细补充信息
4. **file**：读取用户上传的 PDF 财报
5. **message_notify_user**：发送报告给用户（仅当用户明确要求发送文件时）

**重要原则**：
- **所有财务数据查询必须优先使用 gildata-aidata 服务**
- 如 gildata-aidata 数据不完整可辅以 web_search 查询
- 如用户提供 PDF，优先解析 PDF 数据，并用 gildata-aidata 补充衍生数据
- 所有数据获取后需进行交叉验证，确保数据准确性

## gildata-aidata 工具调用指南

### 查询格式说明

gildata-aidata 工具使用自然语言 `query` 参数，调用格式如下：

```bash
finx gildata-aidata <工具名> --body '{"query": "<公司名称> <报告期> <查询内容>"}'
```

### A 股公司常用查询模板

```bash
# 财务报表（三大表）
finx gildata-aidata FinancialStatement --body '{"query": "五粮液 2024 年年报 利润表 资产负债表 现金流量表"}'

# 财务分析指标
finx gildata-aidata FinancialAnalysis --body '{"query": "五粮液 2024 年 毛利率 净利率 ROE ROIC 资产负债率 应收账款周转率 存货周转率"}'

# 主营构成
finx gildata-aidata MainOperIncData --body '{"query": "五粮液 2024 年 分产品 分行业 分地区 收入 成本 毛利率"}'

# 一致预期
finx gildata-aidata ConsensusExpectation --body '{"query": "五粮液 2024 2025 2026 营收 净利润 EPS 一致预期"}'

# 估值数据
finx gildata-aidata StockValueAnalysis --body '{"query": "五粮液 PE PB PS EV 股息率"}'

# 机构评级
finx gildata-aidata InstitutionalRating --body '{"query": "五粮液 机构评级 评级分数"}'

# 业绩预告

# 财报业绩点评参考
```
