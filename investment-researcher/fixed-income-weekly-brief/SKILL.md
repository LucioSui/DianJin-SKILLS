# 固定收益周报生成技能

## 技能概述

本技能面向证券固定收益研究员，自动生成标准化的固定收益市场周报。通过 `gildata-aidata` 专业金融数据服务获取权威数据，覆盖一周内的资金面、利率债、信用债、同业存单、地方债、二级资本债、永续债、ABS、央行操作及重要政策舆情，输出专业、结构化的Markdown格式周报。

与日报不同，周报更侧重于**周度趋势的确认**、**逻辑的连贯性**以及**下周的前瞻性**，强调将一周高频数据串联成债市叙事，并给出明确的配置建议。

## 数据获取流程

### 第一步：获取当前日期

使用 `time` 工具获取当前日期，确定报告覆盖的周期（通常为周一至周五或最近五个交易日）。

### 第二步：资金面周度数据

使用 `finx gildata-aidata MacroIndustryEDB` 获取资金面周度核心指标：

```bash
# 获取DR系列利率周度数据
finx gildata-aidata MacroIndustryEDB --body '{"query": "DR001 DR007 DR014 银行间质押式回购利率 周度"}'

# 获取R系列利率周度数据
finx gildata-aidata MacroIndustryEDB --body '{"query": "R001 R007 R014 质押式回购利率 周度"}'

# 获取SHIBOR利率周度数据
finx gildata-aidata MacroIndustryEDB --body '{"query": "SHIBOR 隔夜 7天 14天 1个月 周度"}'

# 获取同业拆借利率周度数据
finx gildata-aidata MacroIndustryEDB --body '{"query": "同业拆借利率 隔夜 7天 周度"}'
```

## 数据处理与分析

### 周度变动计算

对获取的利率数据进行以下计算：

1. **周度变动**：本周末利率 - 上周末利率（单位：BP）
2. **月度变动**：本周末利率 - 上月末利率（单位：BP）
3. **年初至今变动**：本周末利率 - 上年末利率（单位：BP）

### 期限利差分析

计算关键期限利差及其周度变化：

1. **国债期限利差**：
   - 10Y-1Y 利差及周变动
   - 10Y-2Y 利差及周变动
   - 30Y-10Y 利差及周变动

2. **信用利差**：
   - AAA/AA+/AA 各等级信用利差及周变动
   - 城投债/产业债信用利差分化情况

3. **曲线形态判断**：
   - 根据期限利差变化判断曲线形态（牛陡/牛平/熊陡/熊平）
   - 结合一周内曲线形态演变进行分析

```markdown
## 十三、风险提示

- **货币政策风险**：央行流动性投放不及预期，资金面超预期收敛
- **经济数据风险**：经济数据超预期回暖，基本面修复超预期
- **供给冲击风险**：政府债集中发行，利率债供给压力骤增
- **信用风险**：信用风险事件发酵，引发市场风险偏好下降
- **外部因素**：海外货币政策收紧，中美利差倒挂加深
- **监管政策**：金融监管政策超预期收紧
```

## 数据获取注意事项

1. **数据准确性优先**：如果某些数据无法获取，应明确标注"数据待更新"或"暂无数据"，不要编造数据
2. **多源交叉验证**：对关键数据（如10年期国债收益率）应从多个工具交叉验证
3. **时效性**：优先获取本周末（周五或最近交易日）数据，如无当日数据可使用最近交易日数据并注明
4. **单位统一**：利率统一用百分比(%)，利差和变动用基点(BP)，金额用亿元
5. **专业性**：使用固收研究员常用的专业术语，如"资金面收敛/宽松"、"曲线牛陡/熊平"、"资产荒"、"信用分层"等
6. **周度视角**：周报必须基于本周（通常为周一至周五）的数据与新闻，侧重总结一周的趋势变化，而非单日的波动

## 数据获取失败处理

如果某项数据查询失败：
1. 尝试更换查询关键词
2. 如仍无法获取，在对应表格中标注"数据待更新"
3. 在点评部分说明数据缺失原因
4. 不影响其他已完成部分的数据展示

## 输出要求

1. 报告必须使用标准Markdown格式
2. 表格数据对齐，数字保留合理小数位（利率保留2-3位小数，BP取整数）
3. 点评部分应简洁专业，避免冗长叙述
4. 核心观点必须精炼，便于快速阅读
5. 所有数据需注明数据来源（"数据来源：gildata-aidata"）
6. **报告生成后直接在对话中显示完整内容，无需发送文件**

## 高级功能

### 定时任务支持

本技能支持通过定时任务每周自动生成固收周报。配置建议：
- 执行时间：每周五 18:00-19:00 或周一 09:00-10:00
- 任务描述：自动生成上周固定收益市场周报

### 自定义模板

用户可要求调整报告结构，如：
- 精简版：仅保留核心观点、资金面、利率债、信用债、下周展望五部分
- 详细版：包含全部板块（地方债、二级资本债、永续债、ABS等）
- 定制版：按用户需求调整章节顺序或增减内容

### 历史对比功能

用户可要求添加历史对比维度，如：
- 与上月末/季末/年初数据对比
- 与去年同期数据对比
- 关键指标历史分位数分析

## 技能调用示例

**用户输入：** "生成本周固收周报"

**技能执行：**
1. 获取当前日期，确定报告周期（上周末至本周末）
2. 使用 `gildata-aidata` 服务依次查询：
   - `MacroIndustryEDB` 获取资金面周度利率（DR001/DR007/DR014、R001/R007、SHIBOR）
   - `MostActiveRateBondQuotation` 获取利率债最活跃券周度收益率
   - `BondDailyClosingQuotes` 获取国债期货、信用债、地方债、二级资本债、永续债、ABS周度行情
   - `NCDIssueRate` 获取同业存单周度发行利率
   - `MacroIndustryEDB` 获取央行公开市场操作周度汇总
   - `MacroNewslist` 和 `NewsInfoList` 获取政策舆情周度汇总
3. 计算周度变动、期限利差、信用利差等指标
4. 整理数据填入报告模板
5. 输出完整Markdown格式周报
