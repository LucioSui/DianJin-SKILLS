# 行业比较与配置推荐 (Sector Allocation & Comparison)

## 概述

本技能辅助证券策略研究员进行**行业轮动分析**与**资产配置决策**。通过 gildata-aidata 专业金融数据服务自动化采集行业核心数据，运用多维比较框架（景气、估值、资金、政策），量化评估各行业投资价值，输出标准化的行业配置排序与深度逻辑推演。

## 核心分析框架 (4D 模型)

### 1. 景气维度 (Fundamentals / Momentum) - 40% 权重
* **盈利预测**: 分析师盈利预测上调比例（EPS 上调家数/总家数）。
* **业绩增速**: 营收/净利润同比增速、环比改善情况。
* **高频数据**: 行业高频指标（如汽车销量、水泥价格、半导体出货量、运价指数等）的趋势。
* **景气评分**: 趋势向上且超预期者为高分。

### 2. 估值维度 (Valuation) - 30% 权重
* **绝对估值**: PE (TTM)、PB (LF) 绝对值。
* **相对估值**: PE/PB 在过去 3 年/5 年/10 年的分位数水平（<20% 低估，>80% 高估）。
* **性价比**: PEG (PE / 净利润增速)、风险溢价 (ERP)。
* **估值评分**: 低估值且具备修复弹性者为高分。

### 3. 资金维度 (Liquidity / Flow) - 20% 权重
* **北向资金**: 近期净流入/流出规模及趋势。
* **机构持仓**: 公募基金持仓占比变化（超配/低配幅度）。
* **交易热度**: 融资买入占比、换手率分位数、成交额占比。
* **资金评分**: 资金持续流入且未过度拥挤者为高分。

### 4. 政策与催化 (Catalysts) - 10% 权重
* **政策导向**: 国家战略规划、产业政策支持力度（加分项）。
* **事件催化**: 涨价、重组、技术突破等短期催化剂。

## 数据采集指引

**所有数据查询必须使用 `gildata-aidata` 服务**，具体工具映射如下：

### 估值数据获取
```
finx gildata-aidata IndustryValuation --body '{"industryCode": "行业代码", "date": "查询日期"}'
```
获取行业 PE(TTM)、PB(LF)、总市值、流通市值、股息率等估值指标。

### 资金流向数据获取
```
finx gildata-aidata IndustryCapitalFlow --body '{"industryCode": "行业代码", "startDate": "开始日期", "endDate": "结束日期"}'
```
获取行业每日资金流入流出情况，包括主力净额、超大单、大单等。

### 板块行情排序
```
finx gildata-aidata SectorRank --body '{"sectorType": "行业板块", "sortField": "涨跌幅", "tradeDate": "交易日期"}'
```
获取行业板块列表并按涨跌幅、换手率等指标排序。

### 板块资金流排序
```
finx gildata-aidata SectorFundFlowRank --body '{"sectorType": "行业板块", "sortField": "主力净流入", "tradeDate": "交易日期"}'
```
按资金流向指标对行业板块进行排序，分析主力、散户资金动向。

### 行业财务分析
```
finx gildata-aidata IndustryFinancialAnalysis --body '{"industryCode": "行业代码", "reportType": "最新报告期"}'
```
获取行业财务分析指标，涵盖盈利能力(ROE、毛利率)、偿债能力、成长能力(营收增速、净利增速)等。

### 行业日行情
```
finx gildata-aidata IndustryDailyQuote --body '{"industryCode": "行业代码", "tradeDate": "交易日期"}'
```
获取行业在指定日期的日行情数据，包括收盘价、成交量及涨跌幅。

### 一致预期数据
```
finx gildata-aidata ConsensusExpectation --body '{"stockCode": "成分股代码"}'
```
获取成分股未来三年盈利预测数据，用于计算行业整体盈利预期变化。

### 宏观行业经济数据
```
finx gildata-aidata MacroIndustryEDB --body '{"indicatorCode": "指标代码", "startDate": "开始日期", "endDate": "结束日期"}'
```
查询中国宏观、行业经济数据，如PMI、工业增加值、社零等高频指标。

### 行业研报观点
```
finx gildata-aidata ResearchReport --body '{"industry": "行业名称", "reportType": "行业研究", "limit": 5}'
```
```
finx gildata-aidata IndustryAnalysisViewpoints --body '{"industry": "行业名称", "limit": 10}'
```
获取行业研报及分析观点，用于判断政策导向和市场共识。

## 执行流程

1. **数据采集**:
   * 使用 `gildata-aidata` 的 `SectorRank` 获取最新申万一级行业涨跌幅排序
   * 使用 `IndustryValuation` 获取各行业 PE、PB 及历史分位数
   * 使用 `SectorFundFlowRank` 获取行业资金流向排序
   * 使用 `IndustryFinancialAnalysis` 获取行业盈利能力和成长能力指标
   * 使用 `IndustryCapitalFlow` 获取近期资金流向趋势
   * 使用 `ResearchReport` 或 `IndustryAnalysisViewpoints` 获取行业研报观点
2. **多维打分与排序**:
   * 根据4D框架对各行业进行半定量打分（超配、标配、低配）
   * 构建**行业配置矩阵**：高景气+低估值=黄金配置机会；高景气+高估值=趋势延续但需警惕；低景气+低估值=左侧布局；低景气+高估值=回避
3. **逻辑推演**:
   * 针对推荐超配的行业，撰写核心配置逻辑（景气反转？估值修复？政策驱动？）
   * 针对建议低配的行业，提示潜在风险（业绩不及预期？估值泡沫？资金流出？）
4. **报告生成**:
   * 生成 Markdown 格式的行业配置报告，数据需注明来源和截止日期

## 注意事项

* **数据源唯一性**: 所有行业相关数据必须通过 `gildata-aidata` 服务获取，确保数据权威性和一致性
* **数据时效性**: 确保引用的估值、资金数据为最新可得数据（注明截止日期）
* **逻辑自洽**: 景气、估值、资金三者之间需逻辑自洽，例如高景气通常伴随估值提升，需解释是否透支
* **风格切换**: 注意当前市场的主导风格（成长 vs 价值，大盘 vs 小盘），并在报告中体现风格偏好
* **工具调用规范**: 调用 gildata-aidata 工具时，参数需使用正确的行业代码（申万行业代码）和日期格式
