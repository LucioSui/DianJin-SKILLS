# 宏观经济数据解读简报生成技能

## 概述

本技能用于自动抓取中国核心宏观经济数据，进行预期对比、政策解读和市场影响分析，生成面向投资专业人士的宏观数据解读简报。

## 数据获取策略

### 核心数据源

**Gildata-AIData (gildata-aidata)** 作为主要数据源，提供全方位的宏观经济数据与投研分析工具：

| 数据需求 | 推荐工具 | 说明 |
|---------|---------|------|
| 宏观行业经济数据 | `MacroIndustryEDB` | PMI、CPI、PPI、社融、M2、进出口、工业增加值等 |
| 宏观分析观点 | `MacroeconomicAnalysisViewpoints` | 研报中对GDP、CPI、PMI、政策等宏观维度的分析观点 |
| 宏观舆情 | `MacroNewslist` | 宏观舆情，按区域、情感类型筛选经济预测及事件 |
| 政策会议 | `PolicyMeetingsList` | 国内经济金融领域政策会议动态及内容 |
| 官员讲话 | `OfficialSpeechEventList` | 国内外重要官员讲话及活动信息 |
| 法律法规 | `LawsRegulations` | 法律法规信息，按类型、关键词筛选 |
| 全网舆情 | `NewsInfoList` | 全网舆情，按关键词、来源、情感类型筛选 |
| 市场舆情 | `MarketNewslist` | 市场舆情，按市场类型筛选 |

### 数据获取优先级

1. **宏观核心数据**：优先 `MacroIndustryEDB` 查询 PMI、CPI、PPI、社融、M2 等
2. **宏观观点解读**：优先 `MacroeconomicAnalysisViewpoints` 获取专业分析观点
3. **宏观舆情与政策**：优先 `MacroNewslist` + `PolicyMeetingsList`
4. **网络搜索作补充**：对于市场预期值、最新政策细节等，使用 `web_search` 补充

## 执行流程

### 第一阶段：数据抓取与整理

1. **搜索最新宏观数据**

   **使用 gildata-aidata 查询**：
   ```
   调用 gildata-aidata MacroIndustryEDB：
   query: "PMI 制造业 非制造业 最新"
   query: "CPI PPI 通胀 最新"
   query: "社融 M2 M1 金融数据 最新"
   query: "工业增加值 固定资产投资 社零 最新"
   query: "进出口 贸易顺差 最新"
   ```

   **使用 web_search 补充**：
   - 搜索关键词示例："中国 PMI CPI PPI 最新数据 市场预期"、"中国 社融 M2 最新 预期"
   - 优先从国家统计局、人民银行、海关总署等官方渠道或权威财经媒体获取数据
   - 重点获取市场预期值（共识预期/中值）

2. **提取核心指标**
   重点关注以下指标（如有发布）：

   | 指标类别 | 核心指标 |
   |---------|---------|
   | 景气度 | 制造业PMI、非制造业PMI、综合PMI |
   | 通胀 | CPI同比/环比、PPI同比/环比、核心CPI |
   | 金融 | 社融增量、新增人民币贷款、M2同比、M1同比 |
   | 实体经济 | 工业增加值同比、固定资产投资同比、社会消费品零售总额同比 |
   | 外贸 | 进出口总额同比、出口同比、进口同比、贸易顺差 |
   | 其他 | 城镇调查失业率、房地产投资/销售数据 |

3. **对比市场预期**
   - 搜索各指标的市场预期值（共识预期/中值）
   - 计算实际值与预期值的偏差（高于预期/低于预期/符合预期）

### 第二阶段：数据解读与分析

4. **数据含义解读**
   
   **使用 gildata-aidata 查询宏观分析观点**：
   ```
   调用 gildata-aidata MacroeconomicAnalysisViewpoints：
   query: "GDP 经济增长 最新分析"
   query: "CPI 通胀 通缩 分析观点"
   query: "PMI 景气度 分析观点"
   query: "货币政策 财政政策 分析观点"
   ```
   
   - 对每个核心指标进行简要解读
   - 分析数据反映的经济运行状况（经济扩张/收缩、通胀/通缩压力等）
   - 识别数据中的亮点与异常点（如结构性分化、趋势性变化）
   - 关注同比、环比、两年平均增速等多维度变化

5. **预期偏差分析**
   - 分析实际数据与市场预期产生偏差的原因
   - 判断偏差是暂时性扰动还是趋势性信号
   - 评估偏差对后续经济走势的指示意义

### 第三阶段：政策与市场影响分析

6. **政策传导分析**

   **使用 gildata-aidata 查询政策信息**：
   ```
   调用 gildata-aidata PolicyMeetingsList：
   查询最新政策会议动态
   
   调用 gildata-aidata OfficialSpeechEventList：
   查询重要官员讲话及活动信息
   
   调用 gildata-aidata LawsRegulations：
   查询相关法律法规信息
   ```

   **货币政策维度：**
   - 数据对央行货币政策取向的暗示（宽松/中性/收紧）
   - 对降准、降息、MLF/LPR操作等具体政策工具的影响判断
   - 流动性环境展望（资金面松紧、利率走廊变化）

   **财政政策维度：**
   - 数据对财政政策力度和方向的暗示（积极/稳健/收缩）
   - 对专项债发行、财政支出节奏、减税降费等政策的影响判断
   - 基建投资与财政发力的关联性分析

7. **市场影响评估**

   **使用 gildata-aidata 查询市场舆情**：
   ```
   调用 gildata-aidata MacroNewslist：
   查询宏观舆情，按情感类型筛选
   
   调用 gildata-aidata MarketNewslist：
   查询市场舆情
   ```

   **股市影响：**
   - 整体市场情绪影响（利好/利空/中性）
   - 受益/受损板块分析（如通胀数据利好资源股、社融数据利好金融地产等）
   - 风格偏好判断（成长/价值、大盘/小盘）

   **债市影响：**
   - 利率债走势判断（利多/利空）
   - 收益率曲线形态变化预期（陡峭化/平坦化）
   - 信用利差影响分析

## 注意事项

- **数据时效性**：优先使用最新发布的月度/季度数据，注明数据所属期（如"2026年3月数据"）
- **预期值来源**：如无法获取精确的市场共识预期，可参考主流券商研报预测或财经媒体汇总的预期区间
- **解读客观性**：数据解读应保持客观中立，避免过度推断，区分事实与观点
- **不确定性提示**：对于存在较大不确定性或数据矛盾之处，应明确提示风险
- **数据缺失处理**：如部分指标当日未发布，可在表格中标注"未发布"或跳过该指标
- **gildata-aidata 优先**：所有宏观数据查询优先使用 gildata-aidata 服务，web_search 仅作为补充获取市场预期值

## 输出要求

- 报告使用标准Markdown格式
- 数据表格保持清晰对齐
- 核心结论应简洁明了，适合快速阅读
- 报告长度控制在1500-2500字为宜，聚焦核心信息

## 示例调用

用户输入："今日宏观数据解读"

技能执行：
1. 获取宏观数据：gildata-aidata MacroIndustryEDB（PMI、CPI、PPI、社融、M2 等）
2. 获取宏观观点：gildata-aidata MacroeconomicAnalysisViewpoints
3. 获取政策信息：gildata-aidata PolicyMeetingsList + OfficialSpeechEventList
4. 获取市场舆情：gildata-aidata MacroNewslist + MarketNewslist
5. 补充预期值：web_search 查询市场预期
6. 撰写报告：按模板生成 Markdown 格式宏观数据解读简报
7. 输出报告：直接在对话中显示完整报告内容
