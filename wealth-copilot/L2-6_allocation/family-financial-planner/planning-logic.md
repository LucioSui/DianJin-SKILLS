# 财务指标阈值与规划逻辑

---

## 7 大财务指标标准

### 1. 资产负债率 = 总负债 / 总资产

| 状态 | 范围 | 说明 |
|------|------|------|
| 🟢 健康 | < 50% | 财务结构稳健 |
| 🟡 关注 | 50-70% | 负债偏高，需控制新增负债 |
| 🔴 预警 | > 70% | 债务压力大，优先还债 |

自住房贷例外：如果负债主要是自住房贷，且月供/月收入 < 30%，可适当放宽至 60% 仍为 🟢。

### 2. 流动比率 = 流动性资产 / 流动性负债

| 状态 | 范围 | 说明 |
|------|------|------|
| 🟢 健康 | > 3 | 短期偿债能力充足 |
| 🟡 关注 | 1-3 | 略紧但可管理 |
| 🔴 预警 | < 1 | 短期偿债困难，需立即改善 |

### 3. 紧急储备金月数 = 流动性资产 / 月必要性支出

| 状态 | 范围 | 说明 |
|------|------|------|
| 🟢 健康 | >= 6 个月 | 能应对突发状况 |
| 🟡 关注 | 3-6 个月 | 基本够用但偏紧 |
| 🔴 预警 | < 3 个月 | 紧急储备严重不足 |

职业稳定性修正：
- 公务员/事业单位/大型国企 → 标准可降至 3 个月
- 自由职业/创业者 → 标准应提高至 9-12 个月

### 4. 净储蓄率 = (年收入 - 年支出) / 年收入

| 状态 | 范围 | 说明 |
|------|------|------|
| 🟢 健康 | >= 30% | 储蓄能力强 |
| 🟡 关注 | 20-30% | 可以接受，但提升空间大 |
| 🔴 预警 | < 20% | 储蓄能力弱，财富积累缓慢 |

年龄修正：
- < 30 岁：20% 即可视为 🟢（收入成长期）
- > 50 岁：要求更高，因为临近退休

### 5. 投资回报率 = 年投资收入 / 投资性资产

| 状态 | 范围 | 说明 |
|------|------|------|
| 🟢 健康 | > 5% | 超过通胀+合理溢价 |
| 🟡 关注 | 3-5% | 仅跑赢通胀 |
| 🔴 预警 | < 3% | 未跑赢通胀，资产实际缩水 |

注意：此指标对刚开始投资（投资资产占比低）的年轻家庭参考意义有限。

### 6. 月还贷比 = 月还贷额 / 月收入

| 状态 | 范围 | 说明 |
|------|------|------|
| 🟢 健康 | < 30% | 还贷压力轻 |
| 🟡 关注 | 30-50% | 还贷压力中等，影响生活质量 |
| 🔴 预警 | > 50% | 还贷压力极大，现金流脆弱 |

### 7. 财务自由度 = 年被动收入 / 年支出

| 状态 | 范围 | 说明 |
|------|------|------|
| 🟢 自由 | > 100% | 不工作也能维持生活 |
| 🟡 进阶 | 30-100% | 被动收入可覆盖部分支出 |
| 🔴 起步 | < 30% | 高度依赖工资收入 |

## 现金流推演逻辑

### 关键假设设定

如果用户未明确指定，使用以下默认假设：

```
收入增长率：
- 工资：5%/年（30岁前）、3%/年（30-50岁）、0%（50岁后）
- 租金：2%/年
- 投资收入：与投资回报率挂钩

支出通胀率：
- 日常花销：3%/年
- 教育支出：5%/年
- 医疗支出：5%/年
- 房贷月供：0%（固定）
- 旅行：3%/年

投资报酬率（默认假设）：
- 工作期（有工资收入）：4-6%
- 退休期（只出不进）：3-4%（应更保守）
```

### 财务断裂点分析

当现金流推演显示某年可投资资产 < 0 时，执行以下分析：

```
方法 1：调整储蓄率
  计算：要避免断裂，需每年多存 X 元
  翻译：月支出需减少 X/12 元，或月收入需增加 X/12 元

方法 2：调整投资回报率
  重新调用 AnalyzeCashFlow，将报酬率上调 1-2 个百分点
  对比不同回报率下的资产曲线
  提示：更高回报意味着更高波动，用 MonteCarloSimulate 展示风险

方法 3：调整退休计划
  将退休年龄后延 3-5 年重新推演
  展示延迟退休对资产曲线的影响

方法 4：调整支出预期
  适度降低退休后月支出假设
  展示不同生活标准下的资产曲线

始终同时展示至少 2-3 种路径供用户选择，不替用户做决定。
```

### 教育规划参考数据

```
国内本科：4-8 万/年，4 年 = 16-32 万
国内研究生：5-10 万/年，2-3 年 = 10-30 万
海外本科（美/英/澳）：30-50 万/年，4 年 = 120-200 万
海外研究生：30-60 万/年，1-2 年 = 30-120 万

注意：以上为当前水平，需按 5% 教育通胀率折算到实际入学年份。
```

### 退休规划参考数据

```
退休后月支出估算：
  一般为退休前月支出的 60-80%
  （减少通勤、工作相关支出，但医疗支出增加）

社保养老金估算（粗略）：
  缴费 30 年，退休金约为退休前工资的 40-60%
  具体取决于缴费基数和当地社平工资
```

## 配置方案的报酬率验证

在阶段四中，需要验证推荐配置方案的预期回报率是否能支撑现金流推演的假设：

```
步骤 1：从 GetAssetAllocationPlan 获取推荐配置
步骤 2：用 AnalyzeInvestmentPerformance 计算加权预期回报率
步骤 3：对比
  - 如果加权回报率 >= 现金流推演中的报酬率假设 → 方案可行
  - 如果加权回报率 < 报酬率假设 → 需要提示用户：
    "要达到规划目标，需要承受更高的投资风险（更高权益占比），
    或者调整支出/退休预期"
步骤 4：用 MonteCarloSimulate 展示配置方案的收益概率分布
  - 中位数收益（50% 概率）是否满足假设
  - 悲观情景（25% 概率）下是否会触发财务断裂
```
