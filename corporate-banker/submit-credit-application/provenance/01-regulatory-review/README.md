# 01 - 监管文件研读

## 输入

- 《商业银行授信工作尽职指引》(银监发〔2004〕51号)
- 《贷款风险分类指引》(银监发〔2007〕54号)
- 《商业银行法》第35条贷款审查要求
- 银监会《流动资金贷款管理暂行办法》

## 过程

### 监管要求提取

| 监管文件 | 核心要求 | 映射到Skill功能 |
|---------|---------|----------------|
| 授信工作尽职指引 | 贷前调查必须全面、真实、准确 | 材料完备性评估(步骤3) |
| 贷款风险分类指引 | 授信审批须有完整的客户档案 | 材料盘点(步骤2) |
| 商业银行法第35条 | 严格审查借款人资质 | 前置状态核查(步骤1) |
| 流动资金贷款暂行办法 | 防止重复授信、过度授信 | 案件冲突检查(步骤1) |

### 红线定义

基于监管要求,定义3条一票否决红线:
- R1:必备材料缺失(违反授信工作尽职指引)
- R2:存在活跃审批案件(违反流动资金贷款暂行办法)
- R3:用户未明确确认提交(违反商业银行法审查要求)
