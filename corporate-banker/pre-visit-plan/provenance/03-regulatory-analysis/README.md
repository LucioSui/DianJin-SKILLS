## 监管要求映射
1. **《商业银行授信工作尽职指引》**:
   - 贷前调查要求 → 访前规划必须覆盖客户基本面、近期动态、风险信号
   - 尽职调查完整性 → 必须读取历史尽调报告和拜访记录

2. **《商业银行客户关系管理指引》**:
   - 客户拜访规范 → 每次拜访必须设定明确目标
   - 客户关系维护 → 根据关系深度调整沟通策略

3. **《个人信息保护法》**:
   - 个人数据保护 → 身份证号、银行账号、联系方式必须脱敏
   - 数据最小化原则 → 仅收集和使用拜访必需的客户信息

4. **《商业银行法》第35条**:
   - 贷款审查要求 → 访前规划不得涉及授信审批决策
   - 审慎经营原则 → 不得隐瞒重大风险信号

## 合规红线设计依据
- **R1:禁止虚假营销**:来自《商业银行法》和监管处罚案例
- **R2:禁止泄露客户隐私**:来自《个人信息保护法》第23条
- **R3:禁止隐瞒重大风险**:来自《商业银行授信工作尽职指引》第15条
