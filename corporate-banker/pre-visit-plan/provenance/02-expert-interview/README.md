# 步骤2:专家访谈与经验萃取

## 输入
- 5位资深对公客户经理访谈记录
- 10位信贷审批官客户评估经验
- 50份历史访前规划报告(涵盖不同拜访类型)
- 客户拜访最佳实践指南

## 过程
1. 访谈对公客户经理,了解访前准备的实际痛点和最佳实践
2. 萃取客户经理关注的核心信息(客户基本面、近期动态、风险信号、营销机会)
3. 分析历史访前规划报告的优劣势,提炼成功要素
4. 识别常见踩坑点(行业分类过泛、目标模糊、风险信号埋没、历史数据忽略)
5. 建立拜访策略差异化框架(5种拜访类型 × 4个分析维度)

## 输出
- 访前规划核心信息清单(4个维度,已整合至 SKILL.md 步骤6)
- 拜访策略差异化框架(已整合至 SKILL.md 步骤7)
- 踩坑记录(5条 Gotchas,已整合至 SKILL.md)
- 拜访策略指南(已移至 references/visit-strategy-guide.md)
- 行业分类指南(已移至 references/industry-classification.md)

## 专家经验萃取要点
1. **成功拜访的3个要素**:明确目标、充分准备、个性化策略
2. **风险前置原则**:任何风险信号必须在报告开头显著标注
3. **数据时效性要求**:财务数据超过6个月必须标注可能过时
4. **个性化沟通**:根据客户关系深度调整话术和策略
5. **目标可验证性**:每个主目标必须包含具体动作和可验证结果

- Gotchas #3:风险信号埋没在正文中(来自风险排查案例)
- Gotchas #4:拜访目标过于模糊(来自30份报告分析)
