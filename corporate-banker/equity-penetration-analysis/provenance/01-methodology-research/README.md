# 步骤1:方法论研究

## 输入
- 股权穿透分析经典方法论(递归穿透算法)
- 间接持股比例计算模型(连乘法、多条路径求和)
- 实际控制人认定多维度判定框架
- 关联方识别三圈层理论(法定/人员/隐性)
- 关联交易公允性分析框架(市场比较法、成本加成法)

## 过程
1. 研究股权穿透的标准方法论,确定递归穿透至自然人/终极控制人的算法逻辑
2. 设计间接持股比例计算模型,区分表决权比例和收益权比例
3. 建立实际控制人认定多维度判定框架(直接持股、间接持股、一致行动、表决权委托等)
4. 研究关联方识别的三圈层理论,设计隐性关联方发现技巧
5. 设计关联交易公允性分析框架,量化定价偏离度

## 关键发现
1. 股权穿透必须采用递归算法,直至自然人或国资委等终极控制人
2. 间接持股比例计算需区分控制链连乘和多条路径求和两种情况
3. 实际控制人认定不能仅凭持股比例,必须综合多维度判定
4. 隐性关联方识别是尽调难点,需运用多种技巧(共用地址、电话、财务人员等)
5. 关联交易公允性分析必须量化偏离度,不得仅做定性描述
