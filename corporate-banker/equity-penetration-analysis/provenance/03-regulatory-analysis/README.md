# 步骤3:监管政策分析

## 输入
- 《公司法》第216条关联方定义
- 《商业银行法》第40条关联交易管控
- 银监会《商业银行与内部人和股东关联交易管理办法》
- 证监会《上市公司信息披露管理办法》
- 股权穿透监管要求

## 过程
1. 研读《公司法》第216条,提取关联方定义的法定标准
2. 分析《商业银行法》第40条,明确商业银行关联交易管控要求
3. 研究银监会关联交易管理办法,识别商业银行关联交易的红线和禁止性行为
4. 对照证监会信息披露管理办法,设计股权穿透信息披露标准
5. 建立6条金融合规红线(R1-R6),明确触发条件和处理措施
6. 设计穿透完整性要求(必须至自然人/国资委/终极控制人)

## 输出
- 6条金融合规红线定义(R1-R6,已整合至 SKILL.md Constraints)
- 穿透完整性要求(已整合至 SKILL.md 步骤2)
- 关联方认定法定标准(已整合至 SKILL.md 步骤5)
- 关联交易管控要求(已移至 references/equity-penetration-guide.md)

## 关键发现
1. 《公司法》第216条明确界定关联方范围,包括控股股东、实际控制人、董监高及其近亲属等
2. 商业银行关联交易管控的核心是防范利益输送和风险传染
3. 股权穿透必须穿透至最终自然人或国有出资人,不得以"公众公司"为由提前终止
4. 6条红线(R1-R6)覆盖了逃废债、融资平台、股权代持、资金占用、资产转移、未披露担保等核心风险
5. 关联交易公允性分析必须量化,不得仅做定性描述
