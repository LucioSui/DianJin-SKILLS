# 股权穿透分析指南（Equity Penetration Guide）

本文件提供股权穿透分析的标准方法、复杂场景处理技巧和监管要求，供 equity-penetration-analysis Skill 在步骤2-4执行时查阅。

---

## 一、股权穿透标准方法

### 1.1 穿透原则

**核心原则**：穿透至自然人或国资委等终极控制人，中途不得以"公众公司"为由截止。

**例外情况**：
- 上市公司公众持股部分（通常 < 25%）不再穿透
- 外国政府/主权基金不再穿透

### 1.2 间接持股计算

**直接控制链**：
```
间接持股比例 = 各层持股比例连乘
```

**示例**：
```
张三 → 60% → A公司 → 50% → B公司 → 40% → 目标公司
张三间接持股 = 60% × 50% × 40% = 12%
```

**多条路径**：
```
总持股比例 = 各路径持股比例求和
```

**示例**：
```
路径1：张三 → 60% → A公司 → 45% → 目标公司 = 27%
路径2：张三 → 99%GP → 合伙企业 → 15% → 目标公司 = 14.85%
总持股 = 27% + 14.85% = 41.85%
```

**表决权比例 vs 收益权比例**：
- 表决权比例：按控制链中最小表决权计算
- 收益权比例：按实际持股比例计算
- 两者可能不一致（如有限合伙GP表决权100%，收益权仅1%）

### 1.3 穿透终止条件

| 终止条件 | 说明 | 示例 |
|---------|------|------|
| 自然人 | 终极受益人 | 张三、李四 |
| 国资委/财政部 | 国有出资人 | 国务院国资委、某省财政厅 |
| 上市公司公众持股 | 仅公众持股部分不再穿透 | 持股 < 25% 的公众股东 |
| 外国政府/主权基金 | 不再穿透 | 挪威主权基金、新加坡GIC |

## 二、复杂股权安排处理

### 2.1 有限合伙企业

**核心要点**：
- GP（普通合伙人）拥有管理决策权，通常持份额 1%-5%
- LP（有限合伙人）仅享有收益权，不参与管理
- **穿透重点**：识别GP的实际控制人

**穿透方法**：
```
有限合伙企业 → GP（法人） → 穿透至GP实控人（自然人）
             → LP（法人） → 穿透至LP实控人（自然人）
```

**示例**：
```
XX合伙企业（员工持股平台）
├── GP：张三（99%GP，1%LP） → 实际控制人
└── LP：50名员工（各2%LP） → 仅享有收益权
```

### 2.2 信托/资管计划

**核心要点**：
- 信托计划/资管产品为特殊目的载体（SPV）
- **穿透重点**：识别委托人/受益人

**穿透方法**：
```
信托计划 → 委托人（自然人/法人） → 穿透至实控人
         → 受益人（自然人/法人） → 穿透至实控人
```

**注意**：
- 如信托计划为员工持股平台，需穿透至全部受益人
- 如信托计划为家族信托，需穿透至委托人及受益人

### 2.3 境外SPV

**核心要点**：
- 境外SPV（BVI、开曼、香港等）常用于红筹架构或跨境投资
- **穿透重点**：识别境外实际持有人

**穿透方法**：
```
境外SPV → 境外股东 → 穿透至境外实控人（自然人/外国政府）
```

**注意**：
- 如境外股东为上市公司，需查阅其年报/公告识别实控人
- 如境外股东为外国政府/主权基金，不再穿透

### 2.4 VIE架构（协议控制）

**核心要点**：
- VIE（Variable Interest Entity）通过协议控制而非股权控制
- **穿透重点**：分析协议控制的有效性和稳定性

**穿透方法**：
```
WFOE（外商独资企业） → 协议控制 → VIE实体（内资企业）
                      ↓
                独家技术咨询协议
                独家购买权协议
                股权质押协议
                表决权委托协议
```

**风险评估**：
- 协议控制是否合法有效？（中国法律对VIE态度不明确）
- 协议是否可能终止？（单方解除条款）
- 实控人是否为外籍？（外汇登记是否合规）

## 三、实际控制人认定

### 3.1 控制权判定维度

| 维度 | 分析要点 | 权重 | 评分标准 |
|------|----------|------|---------|
| 直接持股 | 直接持有表决权比例 | 高 | ≥ 50% 绝对控制，30%-50% 相对控制 |
| 间接持股 | 通过子公司/合伙等间接持有 | 高 | 连乘计算，多条路径求和 |
| 一致行动 | 是否存在一致行动协议 | 高 | 有协议则合并计算表决权 |
| 表决权委托 | 是否接受他人表决权委托 | 高 | 委托期限、是否可撤销 |
| 董事会控制 | 能否决定半数以上董事人选 | 中 | 提名董事人数 / 董事会总人数 |
| 经营管理 | 是否实际参与/主导日常经营 | 中 | 是否担任董事长/总经理 |
| 否决权/特殊权利 | 是否持有一票否决权等特殊权利 | 中 | 重大事项否决权 |
| 历史沿革 | 公司是否由该人创立/主导发展 | 低 | 创始人、长期实控人 |

### 3.2 控制类型认定

| 控制类型 | 判定标准 | 示例 |
|---------|---------|------|
| 绝对控制 | 合计表决权 ≥ 50% | 张三持股52% |
| 相对控制 | 表决权 < 50% 但为第一大股东且远超第二大 | 张三持股35%，第二大股东持股10% |
| 协议控制 | 通过VIE协议、一致行动协议实现控制 | 张三通过一致行动协议控制另外15%表决权 |
| 共同控制 | 两人或多人共同构成实际控制人 | 张三（30%）+ 李四（25%）共同控制 |
| 无实际控制人 | 股权高度分散，无单一控制人 | 最大股东持股 < 20%，无一致行动 |

### 3.3 特殊情形处理

**夫妻/父子关系**：
- 通常认定为一致行动人
- 合并计算表决权

**国有企业**：
- 穿透至国资委
- 区分国有独资（100%）、国有控股（> 50%）、国有参股（< 50%）

**员工持股平台**：
- GP通常为实际控制人
- LP仅享有收益权，不参与控制

## 四、监管要求

### 4.1 《公司法》第216条关联方定义

**关联方包括**：
- 控股股东及其控制的企业
- 实际控制人及其控制的企业
- 董监高及其近亲属
- 董监高及其近亲属控制的企业

### 4.2 《商业银行法》第40条关联交易管控

**商业银行关联交易要求**：
- 不得向关联方发放无担保贷款
- 不得为关联方融资提供担保
- 关联交易必须公允，不得损害银行利益

### 4.3 证监会《上市公司信息披露管理办法》

**披露要求**：
- 年度报告中必须披露关联方关系及交易
- 重大关联交易必须单独公告
- 实际控制人变更必须及时公告

## 五、股权穿透常见问题

### Q1：上市公司是否需要穿透？

**答**：上市公司控股股东需要穿透，仅公众持股部分（通常 < 25%）不再穿透。

### Q2：如何识别隐性关联方？

**答**：以下信号提示可能存在隐性关联：
- 同一注册地址
- 工商联系电话/邮箱相同
- 成立时间接近且业务互补
- 企业名称风格相似
- 关键时点成立/注销

### Q3：股权代持如何识别？

**答**：以下情况提示可能存在股权代持：
- 工商登记股东与实际控制人明显不符
- 股东为公司员工但无实际出资能力
- 股东为老年人/未成年人
- 历史沿革中频繁变更股东

### Q4：VIE架构的法律效力如何？

**答**：中国法律对VIE架构态度不明确，存在政策风险。需重点分析：
- 协议控制是否合法有效？
- 协议是否可能终止？
- 实控人是否为外籍？

## 六、信贷场景特殊要求

### 6.1 股权穿透深度

| 授信金额 | 穿透深度 | 说明 |
|---------|---------|------|
| < 1000万 | 标准（3层） | 穿透至直接控股股东 |
| 1000万-5000万 | 深度（全穿透） | 穿透至自然人实控人 |
| > 5000万 | 深度（全穿透）+ 隐性关联方扫描 | 全穿透 + 三圈层关联方识别 |

### 6.2 红线信号核查

每次股权穿透必须核查6条红线（R1-R6），详见 SKILL.md 约束条件章节。

### 6.3 报告输出要求

- 所有数据标注：数据来源 + 数据日期 + 是否最新数据
- 不确定性显式标注：疑似/待核实
- 红线信号必须在报告摘要中显著标注
