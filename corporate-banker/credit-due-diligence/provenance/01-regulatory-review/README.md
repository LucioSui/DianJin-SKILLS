# 步骤1：监管文件研读

## 输入
- 《商业银行授信工作尽职指引》
- 《贷款风险分类指引》（五级分类）
- 《商业银行法》第35条贷款审查要求
- 人民银行《征信业管理条例》
- 银监会《流动资金贷款管理暂行办法》
- 发改委《产业结构调整指导目录》

## 过程
1. 逐条研读上述监管文件，提取与尽职调查相关的核心要求
2. 识别监管红线和禁止性行为
3. 分析监管对贷前调查的强制性要求（如实地调查、交叉验证等）
4. 对照行内信贷政策，确认合规边界

## 输出
- 监管要求清单（见本目录监管要求提取记录）
- 合规红线定义（7条一票否决条件的监管依据）
- 尽调流程框架（9步流程的监管合规性验证）

2026-05-05

## 负责人
AI架构师团队 + 风险管理部

## 关键发现
1. 监管要求尽调必须覆盖企业基本面、财务状况、经营真实性、主体资信四大维度
2. 强调"实质重于形式"原则，要求交叉验证
3. 明确禁止向淘汰类产业新增授信
4. 要求建立完整的审计追踪机制
