# 步骤1:方法论研究

## 输入
- 财务报表分析经典方法论(三表分析框架)
- 杜邦分析法(三因素/五因素拆解)
- 财务指标计算模型(盈利能力、偿债能力、运营效率、成长能力)
- 现金流模式识别理论(8种典型组合)
- 财务粉饰识别方法论(收入端、成本端、利润端、现金流端)

## 过程
1. 研究财务报表分析的标准方法论,确定三表分析框架
2. 设计杜邦分析法拆解模型(三因素/五因素)
3. 建立财务指标计算体系(ROE、ROA、ROIC、毛利率、净利率等)
4. 研究现金流模式识别理论,设计8种典型组合信号
5. 设计财务粉饰识别框架,量化预警信号

## 关键发现
1. 财务分析必须包含盈利质量评估(收现比/净现比),不能仅看利润表
2. 杜邦分析是拆解ROE驱动力的核心工具,必须展示连环替代法计算过程
3. 现金流组合信号识别能有效预警企业资金链风险
4. 8条红线条件覆盖了审计意见、现金流、关联方资金占用、存贷双高、应收异常、在建工程、商誉减值、股权质押等核心风险点
