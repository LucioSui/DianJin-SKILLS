# 财务粉饰识别指南（Fraud Detection Guide）

本文件提供财务粉饰常见手段、识别方法和核查清单，供 financial-report-analysis Skill 在步骤6（财务预警与粉饰识别）时查阅。

---

## 一、收入端粉饰

### 1.1 提前确认收入

**手法**：
- 在商品未交付、服务未完成时确认收入
- 利用"签收即确认"政策，人为加快签收流程
- 跨期调节：将下期收入提前至本期

**识别方法**：
- 应收账款增速 >> 营业收入增速（如应收增速超营收增速50pct以上）
- 第四季度收入占比异常高（如 > 40%）
- 合同负债（预收款项）大幅下降，但收入未同步增长

**核查清单**：
- [ ] 应收账款周转天数是否同比延长 > 30天？
- [ ] 第四季度收入占比是否 > 40%？
- [ ] 合同负债变动是否与收入变动匹配？
- [ ] 前五大客户是否为新客户或关联方？

### 1.2 渠道压货

**手法**：
- 向经销商大量发货，超出实际销售能力
- 提供退货权或延长账期，刺激经销商进货
- 期末集中发货，下期大量退货

**识别方法**：
- 存货周转天数持续延长
- 应收账款增速 >> 营业收入增速
- 销售费用率异常升高（渠道返利、促销费用）

**核查清单**：
- [ ] 经销商库存是否过高（需实地调研）？
- [ ] 销售退回率是否 > 5%？
- [ ] 经销商是否集中于少数几家？

### 1.3 关联方交易虚增

**手法**：
- 向关联方高价销售商品或提供劳务
- 通过关联方实现"体外循环"
- 利用关联方虚构交易

**识别方法**：
- 关联交易占比 > 30%
- 关联方销售价格明显高于市场价
- 关联方应收账款账龄长且未计提坏账

**核查清单**：
- [ ] 关联交易定价是否公允（对比市场价）？
- [ ] 关联方应收账款是否计提充分坏账？
- [ ] 是否存在无商业实质的关联交易？

## 二、成本端粉饰

### 2.1 费用资本化

**手法**：
- 将研发费用资本化（开发阶段支出资本化）
- 将利息费用资本化（在建工程利息）
- 将日常维修费用资本化（计入固定资产）

**识别方法**：
- 开发支出 / 研发投入 > 50%（资本化比例过高）
- 在建工程余额持续增长，但转固比例低
- 固定资产原值增速 > 营业收入增速

**核查清单**：
- [ ] 研发费用资本化比例是否 > 50%？
- [ ] 在建工程工期是否 > 3年未转固？
- [ ] 资本化利息是否合理（对比有息负债规模）？

### 2.2 少计折旧/减值

**手法**：
- 延长固定资产折旧年限
- 降低坏账准备计提比例
- 不计提存货跌价准备或商誉减值

**识别方法**：
- 固定资产折旧年限明显长于同业
- 坏账准备计提比例 < 同业平均水平
- 商誉 / 净资产 > 30%，但未计提减值

**核查清单**：
- [ ] 折旧政策是否与同业一致？
- [ ] 坏账准备计提比例是否 < 同业？
- [ ] 商誉是否进行减值测试？假设是否合理？

## 三、利润端粉饰

### 3.1 非经常性损益平滑

**手法**：
- 通过处置资产、政府补助等非经常性损益调节利润
- 在盈利好的年份多计提减值，在盈利差的年份释放
- 利用公允价值变动损益调节利润

**识别方法**：
- 非经常性损益 / 净利润 > 30%
- 资产处置收益占比高
- 政府补助占净利润比例 > 20%

**核查清单**：
- [ ] 非经常性损益占比是否 > 30%？
- [ ] 资产处置收益是否具有可持续性？
- [ ] 政府补助是否依赖性强？

### 3.2 公允价值操纵

**手法**：
- 利用投资性房地产公允价值模式调节利润
- 利用金融资产公允价值变动调节利润
- 利用并购标的业绩承诺期调节利润

**识别方法**：
- 公允价值变动损益 / 净利润 > 20%
- 投资性房地产公允价值评估假设明显乐观
- 并购标的业绩承诺刚好完成

**核查清单**：
- [ ] 公允价值评估假设是否合理？
- [ ] 并购标的业绩是否精准完成承诺？
- [ ] 是否存在业绩承诺期满后业绩大幅下滑？

## 四、现金流端粉饰

### 4.1 将筹资现金流包装为经营现金流

**手法**：
- 利用票据贴现、保理等融资行为包装为经营现金流
- 通过关联方代垫款项虚增经营现金流
- 延迟支付供应商款项，临时改善经营现金流

**识别方法**：
- 经营现金流中"收到的其他与经营活动有关的现金"异常高
- 应付票据大幅增加
- 经营活动现金流与净利润背离

**核查清单**：
- [ ] "收到的其他与经营活动有关的现金"是否异常？
- [ ] 应付票据是否与经营规模匹配？
- [ ] 经营现金流改善是否可持续？

### 4.2 存贷双高

**手法**：
- 货币资金高但同时借有大量短期有息负债
- 可能原因：资金受限（保证金、冻结资金）、资金被挪用、虚增货币资金

**识别方法**：
- 货币资金 / 总负债 > 20%
- 同时有息负债 / 总资产 > 40%
- 利息收入 / 货币资金 < 活期存款利率（说明资金可能受限或虚构）

**核查清单**：
- [ ] 货币资金中受限资金占比多少？
- [ ] 利息收入是否与货币资金规模匹配？
- [ ] 是否存在大额资金往来关联方？

## 五、Benford定律辅助验证

### 5.1 原理

Benford定律（首位数字定律）指出，在自然形成的财务数据中，首位数字为1的概率约为30.1%，为2的概率约为17.6%，依此类递减。

### 5.2 应用场景

- 营业收入、营业成本、费用等科目的金额
- 发票金额、报销金额
- 应收账款、应付账款明细

### 5.3 识别方法

- 提取重大科目（如营业收入、营业成本）的首位数字
- 统计各数字（1-9）的分布频率
- 与Benford定律理论分布对比
- 如偏差 > 10%，须进一步核查

### 5.4 注意事项

- Benford定律仅适用于自然形成的数据，不适用于人为设定的数据（如定价、阈值）
- 样本量须 > 1000条才具有统计意义
- 偏差 ≠ 造假，仅作为异常信号提示

## 六、财务预警信号快速检查表

### 6.1 收入端预警

- [ ] 应收账款增速 >> 营业收入增速（超50pct以上）？
- [ ] 合同负债大幅下降，但收入未同步增长？
- [ ] 第四季度收入占比 > 40%？
- [ ] 前五大客户集中度 > 50%？

### 6.2 成本端预警

- [ ] 研发费用资本化比例 > 50%？
- [ ] 在建工程工期 > 3年未转固？
- [ ] 折旧政策明显宽松于同业？
- [ ] 坏账准备计提比例 < 同业？

### 6.3 利润端预警

- [ ] 非经常性损益 / 净利润 > 30%？
- [ ] 政府补助 / 净利润 > 20%？
- [ ] 公允价值变动损益 / 净利润 > 20%？
- [ ] 并购标的业绩精准完成承诺？

### 6.4 现金流端预警

- [ ] 净现比 < 0.5 连续2年？
- [ ] "收到的其他与经营活动有关的现金"异常高？
- [ ] 应付票据大幅增加？
- [ ] 存贷双高（货币资金/总负债 > 20% 且有息负债/总资产 > 40%）？

### 6.5 其他预警

- [ ] 频繁变更会计政策/估计？
- [ ] 审计意见为非标意见？
- [ ] 关联方交易占比 > 30%？
- [ ] 其他应收款异常增大？

## 七、核查建议

### 7.1 资料索取清单

- 近3年审计报告及附注
- 前五大客户/供应商清单及交易明细
- 关联方清单及交易明细
- 重大合同（销售、采购、借款）
- 银行对账单（抽查大额往来）
- 存货盘点报告
- 固定资产清单及折旧政策

### 7.2 实地核查要点

- 实地查看生产经营状况
- 抽查仓库存货
- 访谈财务负责人
- 访谈销售/采购负责人
- 查看银行授信使用情况

### 7.3 第三方验证

- 企查查/天眼查：关联方、诉讼、质押
- 税务局：纳税申报数据
- 海关：进出口数据
- 水电费：生产用电用水数据
