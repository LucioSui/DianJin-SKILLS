# 财务指标计算参考（Financial Indicators Calculation）

本文件提供财务指标的标准计算公式、口径说明和信贷场景特殊要求，供 financial-report-analysis Skill 在步骤2-5计算指标时查阅。

---

## 一、盈利能力指标

### 1.1 毛利率

**公式**：
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```

**口径说明**：
- 营业收入：利润表"营业收入"科目（不含营业外收入）
- 营业成本：利润表"营业成本"科目（不含税金及附加、期间费用）

**信贷场景要求**：
- 须连续计算3年（贷前尽调要求5年）
- 须与行业均值对比（数据来源：Wind/同花顺）
- 毛利率变动 > 5pct 须说明原因

### 1.2 核心利润率

**公式**：
```
核心利润率 = (营业收入 - 营业成本 - 销售费用 - 管理费用 - 研发费用 - 财务费用) / 营业收入 × 100%
```

**口径说明**：
- 三费：销售费用 + 管理费用 + 研发费用 + 财务费用
- 不含资产减值损失、信用减值损失、公允价值变动损益

**评价标准**：
- > 15%：优秀
- 10%-15%：良好
- 5%-10%：一般
- < 5%：关注

### 1.3 净利率

**公式**：
```
净利率 = 净利润 / 营业收入 × 100%
```

**口径说明**：
- 净利润：优先使用"归属于母公司股东的净利润"（归母净利润）
- 如使用"综合净利润"，须在报告中注明口径差异

### 1.4 ROE（净资产收益率）

**公式**：
```
ROE = 净利润 / 平均股东权益 × 100%
平均股东权益 = (期初股东权益 + 期末股东权益) / 2
```

**口径说明**：
- 优先使用"扣除非经常性损益后的归母净利润"
- 平均股东权益：必须使用期初期末平均值，不可仅用期末值

**信贷场景要求**：
- 安全阈值：> 8%（制造业）、> 10%（批发零售业）
- ROE < 5% 连续2年须标注"盈利能力弱"

### 1.5 ROA（总资产收益率）

**公式**：
```
ROA = 净利润 / 平均总资产 × 100%
平均总资产 = (期初总资产 + 期末总资产) / 2
```

### 1.6 ROIC（投入资本回报率）

**公式**：
```
ROIC = NOPAT / 投入资本 × 100%
NOPAT = 息税前利润 × (1 - 所得税率)
投入资本 = 有息负债 + 股东权益 - 货币资金
```

**口径说明**：
- 有息负债：短期借款 + 长期借款 + 应付债券 + 租赁负债
- 货币资金：需扣除受限资金（如保证金、冻结资金）

## 二、盈利质量指标

### 2.1 收现比

**公式**：
```
收现比 = 销售商品、提供劳务收到的现金 / 营业收入
```

**评价标准**：
- > 1.0：优秀（说明有预收款项或回款良好）
- 0.8-1.0：良好
- 0.6-0.8：一般
- < 0.6：关注（可能存在虚增收入或回款困难）

**信贷场景要求**：
- 收现比 < 0.6 连续2年须标注"收入质量差"

### 2.2 净现比

**公式**：
```
净现比 = 经营活动产生的现金流量净额 / 净利润
```

**评价标准**：
- > 1.0：优秀（利润含金量高）
- 0.8-1.0：良好
- 0.5-0.8：一般
- < 0.5：关注（利润可能含大量非现金项目）

**信贷场景要求**：
- 净现比 < 0.5 连续2年须标注"利润含金量低"（红线信号）

### 2.3 核心利润占比

**公式**：
```
核心利润占比 = 核心利润 / 利润总额 × 100%
```

**评价标准**：
- > 80%：优秀（盈利可持续）
- 60%-80%：良好
- 40%-60%：一般
- < 40%：关注（盈利依赖非经常性损益）

## 三、偿债能力指标

### 3.1 资产负债率

**公式**：
```
资产负债率 = 总负债 / 总资产 × 100%
```

**安全阈值**：
- 制造业：< 60%
- 批发零售业：< 70%
- 建筑业：< 75%
- 服务业：< 50%

### 3.2 有息负债率

**公式**：
```
有息负债率 = 有息负债 / 总资产 × 100%
有息负债 = 短期借款 + 长期借款 + 应付债券 + 租赁负债
```

**安全阈值**：
- < 40%：安全
- 40%-60%：关注
- > 60%：警示

### 3.3 流动比率

**公式**：
```
流动比率 = 流动资产 / 流动负债
```

**安全阈值**：
- > 1.5：安全
- 1.0-1.5：关注
- < 1.0：警示（短期偿债压力大）

### 3.4 速动比率

**公式**：
```
速动比率 = (流动资产 - 存货) / 流动负债
```

**安全阈值**：
- > 1.0：安全
- 0.8-1.0：关注
- < 0.8：警示

### 3.5 利息保障倍数

**公式**：
```
利息保障倍数 = 息税前利润 / 利息费用
息税前利润 = 利润总额 + 利息费用
```

**安全阈值**：
- > 3：安全
- 2-3：关注
- < 2：警示（可能无法覆盖利息支出）

## 四、运营效率指标

### 4.1 应收账款周转天数

**公式**：
```
应收账款周转天数 = 365 / (营业收入 / 平均应收账款)
平均应收账款 = (期初应收账款 + 期末应收账款) / 2
```

**评价标准**：
- < 60天：优秀
- 60-90天：良好
- 90-120天：一般
- > 120天：关注（回款慢）

**信贷场景要求**：
- 应收账款周转天数同比延长 > 30天须说明原因
- 账龄 > 1年占比 > 30% 须标注"坏账风险高"

### 4.2 存货周转天数

**公式**：
```
存货周转天数 = 365 / (营业成本 / 平均存货)
平均存货 = (期初存货 + 期末存货) / 2
```

**评价标准**：
- < 60天：优秀
- 60-90天：良好
- 90-180天：一般
- > 180天：关注（可能存在滞销）

### 4.3 总资产周转率

**公式**：
```
总资产周转率 = 营业收入 / 平均总资产
```

**评价标准**：
- > 1.0：优秀
- 0.8-1.0：良好
- 0.5-0.8：一般
- < 0.5：关注（资产利用率低）

## 五、杜邦分析指标

### 5.1 经典三因素杜邦

**公式**：
```
ROE = 净利率 × 总资产周转率 × 权益乘数
    = (净利润/营业收入) × (营业收入/总资产) × (总资产/股东权益)
```

**变动归因方法**：连环替代法
1. 基准年 ROE = 净利率₀ × 周转率₀ × 权益乘数₀
2. 替换净利率：净利率₁ × 周转率₀ × 权益乘数₀
3. 替换周转率：净利率₁ × 周转率₁ × 权益乘数₀
4. 替换权益乘数：净利率₁ × 周转率₁ × 权益乘数₁
5. 各因素贡献 = 相邻两次计算差值

### 5.2 五因素杜邦

**公式**：
```
ROE = 税负系数 × 利息负担系数 × 经营利润率 × 资产周转率 × 权益乘数
    = (税后利润/税前利润) × (税前利润/EBIT) × (EBIT/营业收入) × (营业收入/总资产) × (总资产/权益)
```

**更精细的归因**：
- 税负系数：反映税收负担
- 利息负担系数：反映财务杠杆成本
- 经营利润率：反映核心经营盈利能力
- 资产周转率：反映资产利用效率
- 权益乘数：反映财务杠杆水平

## 六、信贷场景专属指标

### 6.1 DSCR（债务偿付覆盖率）

**公式**：
```
DSCR = 可用于还本付息的现金流 / 应还本付息额
可用于还本付息的现金流 = 经营现金流 - 维持性资本开支
```

**安全阈值**：
- ≥ 1.2：安全
- 1.0-1.2：关注
- < 1.0：警示（无法覆盖债务）

### 6.2 有息负债/EBITDA

**公式**：
```
有息负债/EBITDA = 有息负债 / (息税前利润 + 折旧 + 摊销)
```

**安全阈值**：
- < 3×：安全
- 3-5×：关注
- > 5×：警示

### 6.3 关联资金占用率

**公式**：
```
关联资金占用率 = 其他应收款（关联方）/ 净资产 × 100%
```

**安全阈值**：
- < 10%：安全
- 10%-20%：关注
- > 20%：警示
- > 30%：红线（可能存在资金输送）

## 七、指标计算注意事项

### 7.1 口径一致性

- 跨年对比必须统一口径
- 如会计准则变更导致口径变化，须调整历史数据或注明"口径不可比"
- 净利润口径：优先使用"归母净利润"，如使用"综合净利润"须注明

### 7.2 异常值处理

- 如某年数据异常（如疫情、灾害），须在分析中说明
- 不可简单剔除异常年份，须解释原因并做敏感性分析

### 7.3 行业对标

- 行业均值数据来源：Wind/同花顺/证监会行业分类
- 须使用相同行业分类（GB/T 4754-2017 细化至小类）
- 如企业跨行业经营，须按收入占比加权计算行业均值

### 7.4 预测假设

- 涉及预测（如DCF、压力测试）须明确列出所有假设
- 假设须有合理依据（历史趋势、行业预测、宏观经济）
- 须做敏感性分析（如收入增长率 ±5%、毛利率 ±3%）
