# 现金流组合模式指南（Cashflow Pattern Guide）

本文件提供现金流组合模式的识别方法和典型解读，供 financial-report-analysis Skill 在步骤4（现金流深度分析）时查阅。

## 二、各模式详细解读

### 2.1 成长期模式（+ - +）

**特征**：
- 经营现金流为正：主营业务具备造血能力
- 投资现金流为负：持续资本开支（扩产、并购）
- 筹资现金流为正：外部融资支持扩张

**典型案例**：
- 制造业扩建厂房、购买设备
- 科技企业研发投入、并购团队
- 连锁企业开设新店

**信贷关注点**：
1. **扩张是否理性**：
   - 资本开支/折旧摊销 > 1.5（扩张激进）
   - 资本开支/经营现金流 > 80%（高度依赖外部融资）
   
2. **融资成本**：
   - 有息负债增速 > 经营现金流增速
   - 利息保障倍数 < 3
   
3. **投产进度**：
   - 在建工程工期是否合理？
   - 产能利用率是否提升？

**授信建议**：
- 可支持，但须关注投产进度和产能消化
- 建议追加抵押或担保
- 授信期限与项目周期匹配

### 2.2 成熟期模式（+ - -）

**特征**：
- 经营现金流为正：主营业务稳定造血
- 投资现金流为负：适度资本开支（维护性投资）
- 筹资现金流为负：偿还债务、分红、回购

**典型案例**：
- 传统制造业龙头企业
- 消费品行业成熟企业
- 公用事业企业

**信贷关注点**：
1. **债务压力**：
   - 有息负债/EBITDA < 3×（安全）
   - 偿债资金来源是否单一？
   
2. **分红是否过度**：
   - 分红率 > 80%（可能影响偿债能力）
   - 分红与经营现金流匹配度
   
3. **行业前景**：
   - 行业是否面临替代风险？
   - 企业是否缺乏新增长点？

**授信建议**：
- 优质客户，可给予较低利率
- 授信期限可适度延长
- 关注行业周期变化

### 2.3 收缩期模式（+ + -）

**特征**：
- 经营现金流为正：主营业务仍具造血能力
- 投资现金流为正：变卖资产、收回投资
- 筹资现金流为负：偿还债务

**典型案例**：
- 企业战略转型，退出非核心业务
- 企业面临流动性压力，被迫变卖资产
- 企业优化资产结构

**信贷关注点**：
1. **为何变卖资产**：
   - 主动战略调整？还是被迫自救？
   - 变卖资产是否为核心资产？
   
2. **是否可持续**：
   - 可变现资产是否充足？
   - 经营现金流是否足以覆盖债务？
   
3. **资金用途**：
   - 变卖资产所得是否用于偿还债务？
   - 是否存在资金挪用？

**授信建议**：
- 谨慎支持，须核实变卖资产原因
- 建议缩短授信期限
- 要求提供还款计划

### 2.4 烧钱期模式（- - +）

**特征**：
- 经营现金流为负：主营业务失血
- 投资现金流为负：持续资本开支
- 筹资现金流为正：高度依赖外部融资

**典型案例**：
- 初创期科技企业
- 互联网平台企业
- 生物医药研发企业

**信贷关注点**：
1. **融资渠道是否可持续**：
   - 是否有知名投资机构背书？
   - 是否有IPO计划？
   - 现有资金可支撑多久？
   
2. **商业模式是否可行**：
   - 单位经济模型是否成立？
   - 客户获取成本 vs 客户终身价值
   
3. **烧钱速度**：
   - 经营现金流/营业收入（亏损率）
   - 货币资金/月均经营现金流（可支撑月数）

**授信建议**：
- 高风险，建议仅提供短期授信
- 要求强担保或抵押
- 建议分期放款，设定里程碑

### 2.5 危险期模式（- + +）⚠️

**特征**：
- 经营现金流为负：主营业务失血
- 投资现金流为正：变卖资产
- 筹资现金流为正：借钱续命

**典型案例**：
- 经营恶化企业
- 资金链断裂企业
- ST/*ST企业

**信贷关注点**：
1. **经营为何失血**：
   - 行业周期？还是企业自身问题？
   - 是否可逆转？
   
2. **变卖资产是否为核心资产**：
   - 如变卖核心资产，企业是否还有持续经营能力？
   
3. **融资渠道是否枯竭**：
   - 是否已无法获得股权融资？
   - 银行授信是否已用满？

**授信建议**：
- ⚠️ 高风险，建议暂停授信
- 如已授信，建议加强贷后管理
- 要求追加担保或提前还款

### 2.6 特殊期模式（+ + +）

**特征**：
- 经营现金流为正：主营业务造血
- 投资现金流为正：变卖资产
- 筹资现金流为正：大量融资

**典型案例**：
- 企业准备大额并购
- 企业建立资金池
- 企业应对不确定性（如疫情储备资金）

**信贷关注点**：
1. **资金用途**：
   - 是否有明确的大额资金用途？
   - 是否存在资金池或资金挪用？
   
2. **融资规模是否合理**：
   - 融资规模与资金用途是否匹配？
   - 是否存在过度融资？

**授信建议**：
- 须核实资金用途
- 建议受托支付，确保资金用于约定用途
- 加强贷后资金流向监控

### 2.7 衰退期模式（- - -）

**特征**：
- 经营现金流为负：主营业务失血
- 投资现金流为负：仍在投资（可能无法停止）
- 筹资现金流为负：偿还债务（可能被迫）

**典型案例**：
- 行业衰退期企业
- 产能过剩行业企业
- 高杠杆企业去杠杆

**信贷关注点**：
1. **行业是否可逆转**：
   - 周期性衰退？还是结构性衰退？
   
2. **资金链是否断裂**：
   - 货币资金/短期债务 < 0.5？
   - 是否已出现逾期？
   
3. **资产变现能力**：
   - 担保物是否易变现？
   - 变现折扣多少？

**授信建议**：
- ⚠️ 高风险，建议暂停新增授信
- 如已授信，建议制定退出计划
- 要求提前还款或追加担保

## 三、现金流分析注意事项

### 3.1 季节性因素

- 某些行业经营现金流具有季节性（如农业、零售业）
- 须对比同期数据，不可简单环比
- 建议看全年数据，而非单季度

### 3.2 一次性因素

- 大额资产处置、政府补助、诉讼赔偿等一次性因素
- 须剔除一次性因素，看常态现金流
- 在报告中注明"剔除一次性因素后"

### 3.3 会计政策变更

- 现金流量表分类标准变更（如票据贴现分类）
- 须调整历史数据或注明"口径不可比"

### 3.4 合并报表范围变化

- 并购/处置子公司导致合并范围变化
- 须注明"同口径对比"
- 如变化重大，建议单独分析母公司报表

## 四、现金流压力测试

### 4.1 测试场景

| 场景 | 假设 | 测试目的 |
|------|------|---------|
| 基准场景 | 历史趋势延续 | 正常情况下的现金流 |
| 轻度压力 | 收入下降10%，毛利率下降2pct | 轻微波动下的偿债能力 |
| 中度压力 | 收入下降20%，毛利率下降5pct | 中等冲击下的偿债能力 |
| 重度压力 | 收入下降30%，毛利率下降8pct | 极端情况下的偿债能力 |

### 4.2 测试指标

- DSCR（债务偿付覆盖率）是否 > 1.2？
- 货币资金/短期债务是否 > 0.5？
- 经营现金流是否足以覆盖利息支出？
- 是否需要外部融资？融资渠道是否畅通？

### 4.3 测试报告格式

```markdown
## 现金流压力测试结果

### 基准场景
- DSCR：1.5（安全）
- 货币资金/短期债务：0.8（安全）
- 经营现金流/利息支出：4.2（安全）

### 轻度压力（收入-10%，毛利率-2pct）
- DSCR：1.2（临界）
- 货币资金/短期债务：0.6（关注）
- 经营现金流/利息支出：2.8（安全）

### 中度压力（收入-20%，毛利率-5pct）
- DSCR：0.9（⚠️ 无法覆盖债务）
- 货币资金/短期债务：0.4（⚠️ 流动性紧张）
- 经营现金流/利息支出：1.5（关注）

### 重度压力（收入-30%，毛利率-8pct）
- DSCR：0.6（⚠️ 严重不足）
- 货币资金/短期债务：0.2（⚠️ 可能断裂）
- 经营现金流/利息支出：0.8（⚠️ 无法覆盖利息）

### 结论
- 企业在基准场景和轻度压力下具备偿债能力
- 中度压力下DSCR < 1.0，建议追加担保或缩短授信期限
- 重度压力下流动性可能断裂，建议制定应急预案
```
